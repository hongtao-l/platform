## 一、设计概述

- **原型设计**：基于现有 `views/liveview/` 代码重构
- **UI框架**：Vant 4 + 自定义 SCSS
- **样式复用**：`styles/variables.scss` 变量体系
- **设计原则**：统一布局框架适配多种设备类型，组件化复用避免重复代码

### 2.2 页面组成

| 区域 | 内容 | 高度 |
|------|------|------|
| 顶部栏 | 返回按钮 + 设备名称 + 信号强度 + 设置按钮 | 44px |
| 视频画面 | 单路实时视频 full-width，底部叠加清晰度/全屏控件 | flex:1 |
| 底部功能区 | 功能按钮网格（2行×4列） | ~160px |
| PTZ面板 | 3x3方向键 + 变焦滑块，条件展开 | ~200px |

### 2.3 顶部栏

```
位置：视频画面顶部叠加
背景：linear-gradient(180deg, rgba(0,0,0,0.55) 0%, transparent 100%)
布局：flex row, items-center
  - 左侧：返回箭头 (van-icon arrow-left, 22px, #fff)
  - 中间：设备名称 (font-size:15px, font-weight:600, #fff, flex:1)
  - 右侧1：信号强度 (wifi图标 + 文字, 11px)
  - 右侧2：设置齿轮 (van-icon setting-o, 20px, #fff)
```

### 2.4 视频画面区

```
位置：flex:1，黑色背景
画面：全宽适应，保持16:9比例
画面底部叠加层（半透明）：
  - 左侧：清晰度选择（标清 SD / 高清 HD / 超清 UHD），当前选中高亮蓝色
  - 右侧：全屏按钮 (expand icon)
```

### 2.5 底部功能区

```
背景：#1A1A2E（深色面板）
布局：2行 × 4列 网格
每个功能按钮：
  - 图标区：36×36px 圆形，半透明白底 rgba(255,255,255,0.08)
  - 标签：9px, rgba(255,255,255,0.6), 居中对齐
  - 激活态：图标背景变主题色半透明

标准摄像机功能按钮（第一行）：
  语音对讲 | PTZ控制 | 回放 | 更多
标准摄像机功能按钮（第二行）：
  录制 | 子设备 | 截图 | 补光

「更多」展开后替换第二行为：
  预置位 | 警笛 | 画面翻转 | 收起
```

### 2.6 PTZ 控制面板

```
位于底部功能区和视频画面之间
背景：#1A1A2E
内容：
  - 标题「云台控制」14px #fff 左对齐
  - 3x3 方向网格（120×120px，居中）
    - 上/下/左/右：箭头图标，圆角方块 bg#2A2A40
    - 中心：+ 图标，主题蓝色 bg $primary-color
  - 变焦滑块行：
    - 左侧「变焦」标签
    - el-slider (1-10)
    - 右侧「5x」数值

点击PTZ按钮 → 面板展开（slide-down动画）
再次点击PTZ或点击其他区域 → 面板收起
```

### 3.3 多屏交互

| 交互 | 触发 | 行为 |
|------|------|------|
| 点击某屏 | 单击任意镜头画面 | 该屏变为活跃状态，蓝色边框高亮 |
| 进入全屏 | 点击活跃屏的全屏按钮 | 该镜头全屏展示，支持缩放/平移 |
| 退出全屏 | 全屏模式点击返回/手势 | 回到多屏并列视图 |
| 单屏缩放 | 活跃屏内双指缩放/滚轮 | 画面缩放（1x-8x），显示全景小地图 |
| 清晰度切换 | 点击清晰度选项 | 仅切换当前活跃屏的清晰度 |

### 3.4 全景小地图（仅单屏模式）

```
位置：右上角浮动
尺寸：80px 宽，高度按 32:9 比例
背景：rgba(0,0,0,0.55) + backdrop-filter: blur(4px)
内容：
  - 缩小版全景画面缩略图
  - 橙色矩形框标示当前视口位置
  - 底部标签「全景」9px
交互：
  - 可拖拽移动小地图位置
  - 点击小地图某位置 → 视口平移到对应区域
  - 缩放级别 < 1.05 时自动隐藏
```

## 四、页面布局框架（通用）

### 4.1 三段式布局

```
┌──────────────────────────────────┐
│ 顶部栏 (44px, 绝对定位, z-index:10) │
├──────────────────────────────────┤
│                                  │
│   视频画面区 (flex:1, 可滚动)       │
│   - 单屏：一个视频占满             │
│   - 多屏：N个视频等分纵向空间       │
│                                  │
├──────────────────────────────────┤
│ 底部功能区 (~160-200px)            │
│   - 功能按钮网格                   │
│   - 条件面板（PTZ等）              │
└──────────────────────────────────┘
```

### 4.2 通用顶部栏组件

| 元素 | 位置 | 样式 |
|------|------|------|
| 返回按钮 | 左侧 padding-left:14px | van-icon arrow-left 22px #fff |
| 设备名称 | 中间 flex:1 | 15px font-weight:600 #fff |
| 信号/缩放指示 | 右侧-1 | 11px #fff |
| 设置按钮 | 右侧-2 padding-right:14px | van-icon setting-o 20px #fff |

### 4.3 通用底部功能区

```
背景色：#1A1A2E
内边距：12px 8px
安全区：env(safe-area-inset-bottom)

功能按钮网格：
  - 默认展示 2行×4列
  - 每个按钮 flex-column 垂直排列
  - 图标 36px 圆形 + 标签 9px
  - 「更多」展开第三行（功能更多时）

按钮状态：
  - 默认：图标 bg rgba(255,255,255,0.08)
  - 激活：图标 bg 主题色半透明（如 rgba(26,115,232,0.35)）
  - 按下：transform scale(0.92)
```

## 七、交互规范

### 7.1 画面交互

| 操作 | 手势 | 效果 |
|------|------|------|
| 缩放（单屏广角） | 双指捏合/滚轮 | 画面 1x-8x 缩放 |
| 平移（单屏广角） | 单指拖动 | 画面平移 |
| 双击（单屏广角） | 双击画面 | 切换 1x ↔ 2x |
| 全屏切换 | 点击全屏按钮 | 进入/退出全屏 |
| 切换活跃屏 | 单击某画面 | 蓝色边框高亮 |
| 返回 | 点击返回按钮 | 返回上一页 |

### 7.2 面板动画

| 面板 | 动画 | 时长 |
|------|------|------|
| PTZ面板 | slide-down / slide-up | 250ms |
| 子设备面板 | van-popup bottom slide | 300ms |
| 更多功能行 | height expand | 200ms |

## 八、响应式

- 竖屏优先设计（手机端）
- 视频画面16:9比例适配不同屏幕宽度
- 底部功能区：按钮间距自适应（space-around）
- 横屏自动全屏（通过 CSS media query + 路由行为控制）
- 安全区适配：底部 padding-bottom: env(safe-area-inset-bottom)

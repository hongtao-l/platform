# 物模型能力库 - 基础能力管理

## 一、背景与目标

| 项目 | 描述 |
| :--- | :--- |
| **背景** | 对于复用性高、结构相对复杂的物模型能力，需要抽象为**基础能力**，以便添加到各个标准能力模块中复用 |
| **目标** | 支持运营/开发者创建、编辑、删除两类基础能力：定时任务、区域设置 |
| **适用范围** | 全品类设备（IPC、电工照明、传感安防等） |

---

## 二、角色权限

| 角色 | 权限 |
| :--- | :--- |
| 平台管理员 | 增删改查所有基础能力 |
| 厂商/运营商用户 | 仅支持创建产品是调用已添加至标准能力中的基础能力 |

---

## 三、页面入口

| 路径 | 操作 |
| :--- | :--- |
| 生态平台 → 物模型管理 → 能力库 → 基础能力 Tab | 点击“新增基础能力”按钮 |

## 四、新增【基础能力】- 表单定义

### 4.1 通用字段（所有基础能力必填）

| 字段名 | 标识符 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| 能力标识 | `ability_id` | string | ✅ | 全局唯一，仅支持字母开头，后续可包含字母/数字/下划线，如 `TimeTask` |
| 能力名称 | `ability_name` | string | ✅ | 中文名称，如“定时任务”、“区域设置” |
| 能力描述 | `description` | string | ❌ | 说明此能力的用途，如“平台通用物模型能力” |

### 4.2 能力配置（按类型差异化）

#### 类型一：定时任务（`TimeTask`）

| 配置项 | 数据类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| 任务ID | int | 系统自动生成 | 唯一标识每个定时任务实例，用户不可编辑 |
| 任务开关 | bool | `true`（开启） | 决定是否启用此定时任务 |
| 任务开始时间 | int | 0 | 单位：秒；表示从00:00开始的偏移秒数（范围0~86399） |
| 任务结束时间 | int | 86399 | 单位：秒；范围0~86399，必须 ≥ 开始时间 |
| 重复周期 | array[string] | `["mon","tue","wed","thu","fri","sat","sun"]` | 多选；可选值见下表 |

**重复周期可选值**：

| 显示文案 | 编码值 |
| :--- | :--- |
| 周一 | `mon` |
| 周二 | `tue` |
| 周三 | `wed` |
| 周四 | `thu` |
| 周五 | `fri` |
| 周六 | `sat` |
| 周日 | `sun` |

**扩展示意**：一个定时任务能力可包含**多个任务实例**（如“早上开灯”、“晚上关灯”），每个实例独立拥有上述配置项。

---

#### 类型二：区域设置（`AreaSet`）

| 配置项 | 数据类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| 区域ID | int | 系统自动生成 | 唯一标识，用户不可见 |
| 坐标点 | array[object] | `[]`（空） | 每个坐标点包含 `{lat, lng}`，至少3个点构成多边形 |
| 能力配置 | enum | `"四边形"` | 选项：`"四边形"` / `"六边形"`（单选） |
| 其他辅助能力 | string | 空 | 不推荐使用，预留字段 |

**坐标点格式示例**：
```json
{
  "region_id": 1,
  "points": [
    {"lat": 39.9042, "lng": 116.4074},
    {"lat": 39.9142, "lng": 116.4174},
    {"lat": 39.8942, "lng": 116.4274}
  ]
}

### 4.3 标准能力添加除自定义属性、服务、事件外，可多选基础能力
点击【添加标准能力】按钮，弹窗新增一个“基础”类型，点击可多选平台已配置的基础能力

## 六、产品开发模块 - 【设备能力】Tab 改造

### 6.1 改造目标

将原来碎片化的设备能力配置，改为**标准能力形式**，与能力库的交互体验保持一致。

### 6.2 新交互形式

| 元素 | 说明 |
| :--- | :--- |
| 能力列表 | 表格形式展示已添加的能力，列包含：能力名称、能力标识、能力描述、操作 |
| 添加按钮 | "添加标准能力" → 弹出从能力库选择能力的弹窗 |
| 编辑按钮 | 编辑当前能力实例的具体参数（如宠物侦测开关的默认值） |
| 删除按钮 | 从当前设备移除该能力（不删除能力库） |

### 6.3 与能力库的关系

| 操作 | 行为 |
| :--- | :--- |
| 添加能力 | 从能力库复制一份配置到设备；设备可独立修改参数（如定时时间） |
| 编辑设备内能力 | 仅影响当前设备；不影响能力库中的模板 |
| 删除设备内能力 | 仅从当前设备移除；能力库不受影响 |

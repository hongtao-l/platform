# 全能AI A/B Test — 页面设计说明

## 产品设计

- 原型设计：`design/platform/abtest-list.html`、`design/platform/abtest-dashboard.html`
- UI设计：基于 iot-platform 现有 Element Plus 风格，复用 `design/platform_pkg/common.css` 变量体系

## 页面结构

### 页面一：实验列表页

**文件**：`design/platform/abtest-list.html`

**路由**：`/abtest`

**布局**：左侧导航 + 右侧主内容区

**页面组成**：

| 区域 | 内容 |
|------|------|
| 左侧导航 | 复用现有侧边栏，新增「A/B Test」菜单项 |
| 顶部操作栏 | 面包屑导航 + 【创建实验】按钮 |
| 数据表格 | 实验列表，字段见下方 |
| 创建/编辑抽屉 | 右侧滑出，宽度 560px |

**表格字段**：

| 字段 | 说明 | 组件 |
|------|------|------|
| 实验名称 | 加粗显示 | 纯文本 |
| 实验Key | 等宽字体 | 纯文本 |
| 目标分群 | 用户分群名 | 纯文本 |
| 变体A/B URL | 两行域名显示，超出省略 | 纯文本 |
| 流量 | A 50% / B 50% | 纯文本 |
| 状态 | 运行中/已暂停/已结束 | Badge标签 |
| 创建时间 | 日期 | 纯文本 |
| 操作 | 编辑/暂停/数据/删除 | Button组合 |

**创建/编辑抽屉字段**：

| 字段 | 组件 | 必填 | 说明 |
|------|------|------|------|
| 实验名称 | Input | 是 | placeholder: "如：全能AI落地页v1 vs v2" |
| 实验Key | Input(monospace) | 是 | 系统唯一标识，编辑时置灰不可改 |
| 目标分群 | Select | 是 | 全部用户/新注册用户/高价值用户/活跃用户/沉默用户 |
| 变体A URL | Input(monospace) | 是 | 两列并排 |
| 变体B URL | Input(monospace) | 是 | 两列并排 |
| 流量分配 | Slider | — | 默认50/50，拖动调节，实时显示百分比 |
| 备注 | Textarea | 否 | 实验目的和假设说明 |

**交互说明**：
- 点击【创建实验】→ 清空表单，标题「创建实验」
- 点击【编辑】→ 回填数据，标题「编辑实验」，Key 置灰
- 点击【暂停】→ 确认弹窗「确定暂停该实验？暂停后所有用户将看到默认版本。」
- 点击【恢复】→ 直接恢复，状态变为运行中
- 已结束实验显示【应用B(胜出)】按钮 → 确认后将变体URL写入默认配置
- 点击【数据】→ 跳转数据看板页

### 页面二：数据看板页

**文件**：`design/platform/abtest-dashboard.html`

**路由**：`/abtest/:id`

**布局**：左侧导航 + 右侧主内容区

**页面组成**：

| 区域 | 内容 |
|------|------|
| 顶部信息栏 | 面包屑 + 实验状态标签 + 实验基本信息（分群、流量、运行天数） |
| 指标卡片行 | 4个对比卡片，A/B数值 + 差异百分比 |
| 转化漏斗区 | 左右两列漏斗对比 + 底部柱状图 |
| 结论判断区 | 3个条件检查项 + 胜出结论横幅 |

**指标卡片（4个）**：

| 指标 | A值 | B值 | 差异 |
|------|------|------|------|
| 页面曝光(UV) | 1,240 | 1,180 | B -4.8% |
| 套餐选择率 | 30.6% | 35.6% | B +16.3% |
| 支付转化率 | 4.2% | 6.6% | B +57.1% |
| 总收入(¥) | ¥514 | ¥842 | B +63.8% |

**转化漏斗**：

每列展示 曝光 → 选择套餐 → 点击支付 → 支付成功，附每步转化率。胜出变体列高亮（蓝色边框+浅蓝背景）。

**结论判断条件**：

| 条件 | 标准 |
|------|------|
| 每组转化事件 ≥ 500 | 达标绿色✓ / 未达标红色✗ |
| 运行天数 ≥ 7 | 达标绿色✓ / 未达标红色✗ |
| 置信度 ≥ 95% | 达标绿色✓ / 未达标红色✗ |

**胜出横幅**：
- 绿色背景，显示胜出变体 + 转化率对比 + 提升幅度 + 置信度
- 右侧【应用胜出变体】按钮

## 组件复用

| 组件 | 来源 | 说明 |
|------|------|------|
| 侧边栏 | 现有 sidebar | 新增 A/B Test 导航项 |
| 表格 | Element Plus el-table | 现有风格 |
| Badge | 自定义 CSS | 4色：运行中(绿)/已暂停(黄)/已结束(灰)/已应用(蓝) |
| Drawer | Element Plus el-drawer | 560px 宽度 |
| Slider | 原生 input[type=range] | 流量分配用 |

## 色彩规范

| 用途 | 色值 |
|------|------|
| 变体A（对照组） | #64748B (灰) |
| 变体B（实验组） | #2563EB (蓝 primary) |
| 胜出标识 | #16A34A (绿 success) |
| 差异-上升 | #16A34A (绿) |
| 差异-下降 | #DC2626 (红) |

## 响应式

- 最小宽度 1280px，低于此宽度出现横向滚动条
- 表格列宽按内容自适应，URL列允许省略号

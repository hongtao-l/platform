## 二、页面一：实验列表页

### 2.1 页面结构

```
┌─────────────────────────────────────────────┐
│ 左侧导航栏                                    │
│ ┌─────────────┐ ┌──────────────────────────┐ │
│ │ 运营管理     │ │ 面包屑：运营管理 / A/B Test │ │
│ │ ├ 套餐管理   │ ├──────────────────────────┤ │
│ │ ├ A/B Test ◀│ │ [创建实验] 按钮           │ │
│ │ └ ...       │ ├──────────────────────────┤ │
│ └─────────────┘ │ 实验数据表格              │ │
│                 │ ┌────────────────────────┐ │ │
│                 │ │ 名称│ Key  │流量│状态..│ │ │
│                 │ │ ai..│ ai_..│50/50│... │ │ │
│                 │ │ ... │ ...  │ ... │ ... │ │ │
│                 │ └────────────────────────┘ │ │
│                 │ 分页器                     │ │
│                 └──────────────────────────┘ │
└─────────────────────────────────────────────┘
```

### 2.2 页面组成

| 区域 | 内容 |
|------|------|
| 左侧导航 | 复用现有侧边栏，运营管理下新增「A/B Test」菜单项 |
| 面包屑 | 运营管理 / A/B Test |
| 顶部操作栏 | 【创建实验】按钮（右对齐，主色调） |
| 数据表格 | 实验列表，字段见下方 |
| 分页器 | 底部分页，每页20条 |
| 创建/编辑抽屉 | 右侧滑出，宽度 560px |

### 2.3 表格字段

| 字段 | 说明 | 样式 |
|------|------|------|
| 实验名称 | 加粗显示 | font-weight: 700 |
| 实验Key | 等宽字体 | font-family: monospace, font-size: 13px |
| 流量 | A 50% / B 50% | 普通文本 |
| 状态 | 运行中/已暂停/已结束 | Badge标签，4色方案 |
| 创建时间 | YYYY-MM-DD | 普通文本 |
| 操作 | 编辑/暂停/数据/删除 | Button组合，按状态显示 |

### 2.4 状态标签（Badge）配色

| 状态 | 背景色 | 文字色 |
|------|------|------|
| 运行中 | #F0FDF4 | #16A34A |
| 已暂停 | #FFFBEB | #D97706 |
| 已结束 | #F3F4F6 | #6B7280 |
| 待启动 | #EFF6FF | #2563EB |

### 2.5 创建/编辑抽屉字段

| 字段 | 组件 | 必填 | 说明 |
|------|------|------|------|
| 实验名称 | el-input | 是 | placeholder: "如：全能AI落地页v1 vs v2" |
| 实验Key | el-input(monospace) | 是 | 唯一标识，编辑时 disabled |
| 流量分配 | el-slider | — | 默认50，实时显示A/B百分比 |
| 备注 | el-input(type=textarea) | 否 | 实验目的和假设说明 |

> 注：变体A/B的具体内容（页面UI、文案、布局等）由APP端原生实现，不在后台配置。

### 2.6 交互说明

| 操作 | 触发 | 行为 |
|------|------|------|
| 创建实验 | 点击【创建实验】按钮 | 清空表单，打开抽屉，标题「创建实验」 |
| 编辑 | 点击行内【编辑】 | 回填数据，打开抽屉，标题「编辑实验」，Key置灰 |
| 启动 | 点击【启动】 | 确认弹窗，确认后状态变为运行中 |
| 暂停 | 点击【暂停】 | 确认弹窗「确定暂停该实验？暂停后所有用户将看到默认版本。」 |
| 恢复 | 点击【恢复】 | 直接恢复，状态变为运行中 |
| 结束 | 点击【结束】 | 确认弹窗，选择胜出变体或标记无胜出 |
| 应用胜出 | 已结束实验点击【应用B(胜出)】 | 确认弹窗，确认后将胜出变体设为默认 |
| 查看数据 | 点击【数据】 | 跳转数据看板页 |
| 删除 | 点击【删除】 | 二次确认，确认后不可撤销 |

## 三、页面二：数据看板页

### 3.1 页面结构

```
┌─────────────────────────────────────────────┐
│ 顶部信息栏                                    │
│ 面包屑 + 实验状态标签 + 流量/运行天数            │
├─────────────────────────────────────────────┤
│ 指标卡片行（4列）                              │
│ [总曝光UV] [总转化UV] [转化率] [总收入]          │
├─────────────────────────────────────────────┤
│ A/B 对比卡片                                  │
│ ┌──────────────┐ ┌──────────────┐            │
│ │ 变体A         │ │ 变体B ★      │            │
│ │ 曝光 1,240    │ │ 曝光 1,180   │            │
│ │ 选择率 30.6%  │ │ 选择率 35.6% │            │
│ │ 转化率 4.2%   │ │ 转化率 6.6%  │            │
│ │ 收入 ¥514     │ │ 收入 ¥842    │            │
│ └──────────────┘ └──────────────┘            │
├─────────────────────────────────────────────┤
│ 转化漏斗对比                                  │
│ A: 曝光1240→选择380→支付52  转化4.2%          │
│ B: 曝光1180→选择420→支付78  转化6.6%  ★       │
│ [柱状图: A/B 各步对比]                       │
├─────────────────────────────────────────────┤
│ 结论判断区                                    │
│ ☑ 每组转化事件 ≥ 500  当前: A:52 B:78  ❌     │
│ ☑ 运行天数 ≥ 7        当前: 5天         ❌     │
│ ☑ 置信度 ≥ 95%       当前: 91%         ❌     │
│ 结论: 数据不足，继续收集                       │
├─────────────────────────────────────────────┤
│ 每日趋势折线图                                │
│ 横轴: 日期  纵轴: 转化率  A线 vs B线           │
└─────────────────────────────────────────────┘
```

### 3.2 指标卡片

| 指标 | A值 | B值 | 差异显示 |
|------|------|------|------|
| 页面曝光(UV) | 1,240 | 1,180 | B -4.8%（红色↓） |
| 套餐选择率 | 30.6% | 35.6% | B +16.3%（绿色↑） |
| 支付转化率 | 4.2% | 6.6% | B +57.1%（绿色↑） |
| 总收入(¥) | ¥514 | ¥842 | B +63.8%（绿色↑） |

### 3.3 结论判断条件显示

| 条件 | 标准 | 达标 | 未达标 |
|------|------|------|------|
| 每组转化事件 | ≥ 500 | ✅ 绿色 | ❌ 红色 + 显示当前数值 |
| 运行天数 | ≥ 7 | ✅ 绿色 | ❌ 红色 + 显示当前天数 |
| 置信度 | ≥ 95% | ✅ 绿色 | ❌ 红色 + 显示当前置信度 |

### 3.4 胜出横幅

- 仅当三条条件全部达标 + 存在显著差异时显示
- 绿色背景 (#F0FDF4)，边框 (#16A34A)
- 内容：胜出变体 + 转化率对比 + 提升幅度 + 置信度 + P值
- 右侧【应用胜出变体】按钮

## 六、响应式

- 最小宽度 1280px，低于此宽度出现横向滚动条
- 表格列宽按内容自适应
- 指标卡片行在小屏时从4列变为2列

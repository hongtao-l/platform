# 订阅制新增付费试用 — 页面设计说明

> 关联需求：[订阅制新增付费试用01ReqAnalysis.md](订阅制新增付费试用01ReqAnalysis.md)
> 设计端：iot-platform（Vue3 + Element Plus）

---

## 1. 改动范围

套餐配置抽屉 → 套餐定价 → 订阅制区域。仅修改订阅制定价中的「试用」配置段。

```
套餐配置抽屉（el-drawer, 640px）
├── 基础信息
├── 选择套餐服务
├── 套餐定价
│   ├── 定价模式 → 订阅制 / 非订阅
│   ├── [订阅制]
│   │   ├── 划线价（¥ / $）
│   │   ├── 允许试用 ← 改动区域
│   │   ├── 订阅售价（¥ / $）
│   │   └── IAP 价格
│   └── [非订阅]
│       └── ...
├── 套餐标签
└── 投放配置
```

## 3. 字段定义

### 3.1 允许试用

| 属性 | 值 |
|------|------|
| 组件 | `el-switch` |
| 绑定字段 | `form.freeTrial` |
| 默认值 | `false` |
| 布局 | label + switch 同行（`.inline-form-item`，flex 布局） |
| 交互 | 开关切换时展开/收起下方试用配置 |

### 3.2 试用类型

| 属性 | 值 |
|------|------|
| 组件 | `el-radio-group` + `el-radio`，`size="small"` |
| 绑定字段 | `form.trialType` |
| 选项 | `free` → 免费试用 / `paid` → 付费试用 |
| 默认值 | `'free'` |
| 显示条件 | `form.freeTrial === true` |

### 3.3 试用售价

| 属性 | 值 |
|------|------|
| 组件 | `el-input`，`type="number"` |
| 前缀 | 固定前缀标签 ¥ 和 $（`.price-prefix` + `.prefix-label`） |
| 绑定字段 | `form.trialPriceYuan` / `form.trialPriceDollar` |
| 默认值 | `''` |
| 显示条件 | `form.freeTrial === true && form.trialType === 'paid'` |
| 布局 | 双列并排（`.price-row` flex），与订阅售价格式一致 |

### 3.4 试用周期

| 属性 | 值 |
|------|------|
| 组件 | `el-input-number`，`:min="1"` |
| 绑定字段 | `form.trialDays` |
| 默认值 | `7` |
| 显示条件 | `form.freeTrial === true` |

### 3.5 过滤条件

| 属性 | 值 |
|------|------|
| 组件 | `el-select` |
| 绑定字段 | `form.trialFilter` |
| 选项 | `never_this` → 从未试用过此套餐 / `never_any` → 从未试用过该服务类型 / `none` → 不限制 |
| 默认值 | `''` |
| 显示条件 | `form.freeTrial === true` |

---

## 4. CSS 定义

| 类名 | 用途 | 说明 |
|------|------|------|
| `.inline-form-item` | label + switch 同行 | flex 布局，`justify-content: space-between`，覆盖默认 `label-position="top"` |
| `.trial-config` | 试用配置容器 | `margin-top: 12px`，内部表单间距 |
| `.price-row` | 双列并排 | `display: flex; gap: 12px`，子元素 `flex: 1` |
| `.price-prefix` | 价格输入带前缀 | `display: flex`，`.prefix-label` 固定前缀样式 |

---

## 5. 状态说明

| 状态 | 触发条件 | 展示内容 |
|------|------|------|
| 试用关闭 | `freeTrial = false` | 仅显示「允许试用」+ 关闭开关 |
| 免费试用 | `freeTrial = true, trialType = 'free'` | 试用类型 + 试用周期 + 过滤条件 |
| 付费试用 | `freeTrial = true, trialType = 'paid'` | 试用类型 + 试用售价（¥/$）+ 试用周期 + 过滤条件 |
| 价格未填 | 付费试用下价格为空 | 通过 `type="number"` 限制输入格式，默认不主动校验 |

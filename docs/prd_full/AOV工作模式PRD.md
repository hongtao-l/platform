## 业务背景

CareCamPro 应用中的电池管理模块需要支持多种工作模式，以满足不同用户的监控需求。AOV（Always-On Video）工作模式是为电池供电的摄像头设计的一种工作模式，旨在在保证续航能力的前提下，提供持续的视频监控能力。

## 名词解释

| 名词 | 解释 |
|-----|------|
| AOV | Always-On Video，持续视频模式 |
| 帧率 | 每秒拍摄的图片数量，单位为 s/帧 |
| 电量阈值 | 触发自动模式切换的电量百分比 |
| 智能模式 | 根据时间计划自动切换工作模式 |
| 事件录像 | 检测到运动或其他事件时触发的录像 |

## 业务流程图

### 工作模式选择流程

```
用户进入电池管理页面
    ↓
显示当前电量和电池状态
    ↓
用户选择工作模式
    ├─→ 省电模式 → 保存配置 → 完成
    ├─→ AOV 模式 → 配置帧率和电量阈值 → 保存配置 → 完成
    ├─→ 常电模式 → 配置电量阈值 → 保存配置 → 完成
    └─→ 智能模式 → 进入计划设置页面
            ↓
        设置默认执行模式
            ↓
        添加/编辑时间计划
            ↓
        保存配置 → 返回电池管理页面 → 完成
```

### 智能模式计划执行流程

```
系统检查当前时间
    ↓
是否在计划时间内？
    ├─→ 是 → 执行对应工作模式
    └─→ 否 → 执行默认工作模式
    ↓
检查电量是否低于阈值
    ├─→ 是 → 自动切换至省电模式
    └─→ 否 → 继续执行当前模式
```

## 详细功能说明

### 1. 电池状态显示

**位置**：电池管理页面顶部

**显示内容**：
- 电池状态标签（良好/中等/低电量）
- 当前电量百分比（如 78%）
- 预计续航时间（如 12 天）
- 电量进度条（可视化显示）

**交互规则**：
- 实时更新电池状态
- 根据电量自动调整状态标签颜色
- 电量低于 20% 时显示警告

### 2. 省电模式

**功能描述**：设备默认休眠，通过事件唤醒录像，续航更长

**工作原理**：
- 设备处于低功耗休眠状态
- 仅在检测到事件（运动、声音等）时唤醒并录制
- 事件结束后自动返回休眠状态

**适用场景**：
- 低频监控场景
- 需要长续航的场景

**配置项**：无额外配置

### 3. AOV 模式

**功能描述**：设备持续运行，每隔一段时间抓拍一张图片，有事件发生时触发告警录像

**工作原理**：
- 设备持续运行，定时抓拍图片
- 检测到事件时触发告警录像
- 可配置拍照间隔和电量阈值

**适用场景**：
- 需要持续监控的场景
- 需要定时快照的场景

**配置项**：

#### 3.1 拍照帧率设置

- **标签**：拍照帧率（数值越小，功耗越高）
- **可选范围**：1s/帧 ~ 10s/帧
- **默认值**：5s/帧
- **步长**：1s
- **交互方式**：下拉选择器
- **功耗提示**：显示"帧率越低，能耗越高"的警告信息
- **保存方式**：用户选择后立即保存

#### 3.2 电量阈值设置

- **标签**：省电模式电量阈值
- **描述**：电量低于此值时，自动切换至省电模式
- **可选范围**：10% ~ 50%
- **默认值**：20%
- **步长**：10%
- **交互方式**：滑块控制
- **显示格式**：实时显示当前选择的百分比
- **保存方式**：用户调整后立即保存

### 4. 常电模式

**功能描述**：设备一直连续录像，电池电量低于设定值时，自动进入省电模式

**工作原理**：
- 设备持续 24 小时录制
- 监控质量最高
- 功耗最高，续航时间最短

**适用场景**：
- 高频监控场景
- 需要完整录像的场景

**配置项**：

#### 4.1 电量阈值设置

- **标签**：省电模式电量阈值
- **描述**：电量低于此值时，自动切换至省电模式
- **可选范围**：10% ~ 50%
- **默认值**：20%
- **步长**：10%
- **交互方式**：滑块控制
- **显示格式**：实时显示当前选择的百分比
- **保存方式**：用户调整后立即保存

### 5. 智能模式

**功能描述**：根据时间计划配置需要使用的工作模式

**工作原理**：
- 系统根据当前时间检查是否有匹配的计划
- 在计划时间内执行对应的工作模式
- 超出计划时间执行默认工作模式

**适用场景**：
- 需要按时间段切换工作模式的场景
- 工作日和休息日需要不同监控策略的场景

**配置项**：

#### 5.1 默认执行模式

- **标签**：默认执行模式
- **描述**：计划外时间段执行的工作模式
- **可选项**：省电模式、AOV 模式、常电模式
- **交互方式**：单选按钮
- **默认值**：省电模式
- **保存方式**：用户选择后立即保存

#### 5.2 时间计划设置

**添加计划**：
- 工作模式选择（单选）：省电模式、AOV 模式、常电模式
- 开始时间：时:分格式（00:00 ~ 23:59）
- 结束时间：时:分格式（00:00 ~ 23:59）
- 重复设置：周一 ~ 周日多选
- 保存方式：用户确认后立即保存

**删除计划**：
- 支持删除已添加的计划
- 删除后立即生效

**示例**：
- 工作日 08:00-22:00 使用 AOV 模式
- 其他时间使用省电模式

### 6. 事件录像时长设置

**功能描述**：设置事件触发时的录像时长

**配置项**：
- **标签**：事件录像时长
- **可选范围**：5 ~ 30 秒
- **默认值**：10 秒
- **步长**：1 秒
- **交互方式**：进度条形式的滑块
- **显示格式**：实时显示当前选择的时长
- **保存方式**：用户调整后立即保存

### 7. 运动停止自动停止录像

**功能描述**：开启后，检测不到运动时立即停止录像

**工作原理**：
- 开启状态下，设备在检测不到运动时立即停止录像
- 即使未达到设定的录像时长也会停止
- 节省存储空间和电量

**配置项**：
- **标签**：运动停止，则停止录像
- **类型**：开关（Toggle）
- **默认状态**：关闭
- **交互方式**：开关按钮
- **保存方式**：用户切换后立即保存

## 业务规则

### 1. 工作模式优先级

- 用户手动选择的工作模式优先级最高
- 智能模式中的计划优先级次之
- 电量阈值触发的自动切换优先级最低

### 2. 电量阈值逻辑

- AOV 模式：电量低于阈值时自动切换至省电模式
- 常电模式：电量低于阈值时自动切换至省电模式
- 省电模式：无自动切换

### 3. 智能模式逻辑

- 在计划时间内执行对应的工作模式
- 超出计划时间执行默认工作模式
- 多个计划重叠时，按最后添加的计划执行

### 4. 事件录像逻辑

- 事件触发时开始录制
- 录制时长由用户设置（5-30秒）
- 若启用"运动停止则停止录像"，检测不到运动时立即停止

## 数据存储

### 配置项存储

所有配置项需要持久化存储，包括：
- 选中的工作模式
- AOV 模式的帧率设置
- 各模式的电量阈值
- 事件录像时长
- 运动停止开关状态
- 智能模式的计划列表

### 存储方式

- 本地存储（localStorage）或设备端存储
- 支持与云端同步

## 交互设计

### 工作模式切换

- 单选按钮形式
- 选中时高亮显示
- 支持展开/收起配置项

### 参数调整

- 滑块：用于连续范围选择（电量阈值、事件时长）
- 下拉选择器：用于离散选项选择（帧率）
- 开关：用于二元状态切换（运动停止）

### 时间选择

- 时间选择器：支持小时和分钟的独立选择
- 周期选择：多选复选框

## 性能要求

- 工作模式切换响应时间 < 500ms
- 参数保存响应时间 < 200ms
- 页面加载时间 < 1s

## 兼容性

- 支持 iOS 12+
- 支持 Android 8+
- 支持横竖屏切换

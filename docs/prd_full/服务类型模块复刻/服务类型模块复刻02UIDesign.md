# 服务类型模块复刻 — 页面设计说明

> 关联需求：[服务类型模块复刻01ReqAnalysis.md](服务类型模块复刻01ReqAnalysis.md)
> 设计端：iot-platform（Vue3 + Element Plus）

---

## 1. 页面结构

```
推荐位运营
├── 服务类型（本页面）
│   ├── 左侧面板：服务分类
│   │   ├── 分类列表（图标 + 名称 + 数量 badge）
│   │   └── [新建分类] → 分类表单弹窗
│   └── 右侧主区域：服务管理
│       ├── 工具栏：[批量移动] [新增服务]
│       └── 表格：选择框 | 服务名称 | 服务ID | 所属分类 | 最后更新 | 操作(修改/删除)
│           ├── [修改] → 修改服务弹窗（含算法参数配置）
│           └── [删除] → 确认弹窗
```

**路由**：`/package/service`，归属于「套餐管理」导航菜单组。

### 2.2 左侧面板 — 服务分类

| 属性 | 值 |
|------|------|

### 2.3 右侧主区域 — 服务管理

#### 工具栏

```
[批量移动]                    [＋ 新增服务]
```

- 「批量移动」：`el-button`，未选中服务时 disabled
- 「新增服务」：`el-button type="primary"`，带 Plus 图标

#### 表格

| 列 | 宽度 | 说明 |
|------|------|------|
| 选择框 | 50px | `el-table-column type="selection"` |
| 服务名称 | min-width 160px | 文本 |
| 服务 ID | 120px | 数字文本居中 |
| 所属分类 | 120px | `el-tag size="small"` |
| 最后更新 | 160px | `YYYY/M/D HH:mm:ss` |
| 操作 | 140px | 修改 / 删除，`el-button type="text"` 或 link 风格 |

## 3. 弹窗设计

### 3.1 新建/编辑分类弹窗

```
┌───────────────────────────────────┐
│  新建分类                     [✕] │
├───────────────────────────────────┤
│                                   │
│  * 分类名称                       │
│  ┌─────────────────────────────┐  │
│  │ 请输入分类名称                │  │
│  └─────────────────────────────┘  │
│                                    │
│  排序号                           │
│  [ - ]  0  [ + ]                 │
│                                    │
├───────────────────────────────────┤
│                    [取消]  [确定]  │
└───────────────────────────────────┘
```

| 属性 | 值 |
|------|------|
| 组件 | `el-dialog`，400px |
| 标题 | "新建分类" / "编辑分类" |
| 分类名称 | `el-input`，必填，placeholder "请输入分类名称" |
| 排序号 | `el-input-number`，默认 0，`:min="0"`，controls-position="right" |
| 校验 | 名称为空时提示 |

| 属性 | 值 |
|------|------|
| 组件 | `el-dialog`，800px |
| 标题 | "创建服务类型" |
| 服务名称 | `el-input`，必填，placeholder "例如：全天云存"，width 100% |
| 服务 ID | `el-input`，必填，限制数字输入，placeholder "例如：300010"，width 100% |
| 所属分类 | `el-select`，默认"未分类"，数据源为分类列表 |
| 物模型配置 | 双栏布局，中间 添加/移除 按钮组 |

#### 3.2.1 物模型双栏选

| 区域 | 说明 |
|------|------|
| 可选物模型 | 左侧面板，header 显示 "可选物模型 N/M"（N=已勾选可选中的数量, M=总数），搜索框 + checkbox-group 列表，max-height 300px + overflow-y auto |
| 操作按钮 | 中间竖排：「添加」→（向右）「移除」←（向左），未勾选时 disabled |
| 已选物模型 | 右侧面板，header 显示 "已选物模型 N/M"，搜索框 + 列表 |
| 搜索 | 两侧搜索框各自过滤本地列表，`el-input` 带 search 图标 |

**添加流程**：左侧勾选 → 点「添加」→ 物模型从可选移入已选列表
**移除流程**：右侧勾选 → 点「移除」→ 物模型从已选移回可选列表

### 3.3 修改服务类型弹窗

与创建弹窗布局相同，额外增加底部「算法参数详情配置」区域：

```
┌───────────────────────────────────────────────────────────┐
│  修改服务类型                                         [✕] │
├───────────────────────────────────────────────────────────┤
│  （与创建弹窗相同的字段 + 物模型双栏选择）                    │
│                                                            │
│  ─── 算法参数详情配置 ─────────────────────               │
│                                                            │
│  ┌ 1106 · 智能告警-宠物检测 ──────────────────────────┐   │
│  │ ┌────────────────────────────────────────────────┐│   │
│  │ │ # │ 事件类型      │ 是否开启  │ 操作          ││   │
│  │ │───│─────────────│─────────│──────────────││   │
│  │ │ 1 │ 宠物检测告警  │ [开关]   │ 删除          ││   │
│  │ │ 2 │ 宠物移动推送  │ [开关]   │ 删除          ││   │
│  │ └────────────────────────────────────────────────┘│   │
│  │ [+ 添加事件]                                      │   │
│  └────────────────────────────────────────────────────┘   │
│                                                            │
├───────────────────────────────────────────────────────────┤
│                                       [取消]     [保存]    │
└───────────────────────────────────────────────────────────┘
```

| 属性 | 值 |
|------|------|
| 组件 | `el-dialog`，800px |
| 标题 | "修改服务类型" |
| 服务 ID | 展示但置灰不可编辑 |
| 算法配置区 | 按已选物模型分组，每组一个可折叠块 |
| 物模型标签 | 显示物模型 ID + 名称 |
| 事件表格 | 3 列：#序号、事件类型（el-input）、是否开启（el-switch）、操作（删除） |
| 添加事件 | `el-button type="text"`「+ 添加事件」，点击在表格新增一行 |

### 3.4 删除确认弹窗

```
┌──────────────────────────────┐
│  提示                    [✕] │
```

## 4. 状态一览

| 状态 | 分类列表 | 服务表格 | 说明 |
|------|------|------|------|
| 正常态 | 展示全部，含数量 badge | 展示全部或按分类筛选 | |
| 空态（无分类） | 仅展示"未分类"，数量=0 | 空表格「暂无数据」| |
| 空态（无服务） | 正常展示分类 | 空表格「暂无数据」| |
| 加载态 | 骨架屏 / v-loading | v-loading | |
| 错误态 | Toast 提示 | Toast 提示 | |

## 5. 弹窗层级

```
主页面
  ├── 新建/编辑分类 (el-dialog, z=2000)
  ├── 创建/修改服务 (el-dialog, z=2000)
  └── 删除确认 (ElMessageBox, z=2000)
```

## 6. 组件清单

| 组件 | Element Plus | 用途 | 所在区域 |
|------|------|------|------|
| 页面容器 | `.page-container` | 整体布局 | 主页面 |
| 按钮 | `el-button` | 新建分类/批量移动/新增服务/添加事件 | 工具栏 + 弹窗 |
| 输入框 | `el-input` | 分类名称/服务名称/服务ID/搜索 | 弹窗 |
| 数字输入 | `el-input-number` | 排序号 | 分类弹窗 |
| 下拉选择 | `el-select` | 所属分类 | 服务弹窗 |
| 复选框 | `el-checkbox` | 物模型勾选 | 物模型双栏 |
| 开关 | `el-switch` | 事件开启/关闭 | 算法配置 |
| 表格 | `el-table` | 服务列表/事件列表 | 主区域 + 弹窗 |
| 对话框 | `el-dialog` | 所有弹窗 | — |
| 消息确认 | `ElMessageBox.confirm` | 删除确认 | — |
| 标签 | `el-tag` | 分类数量 badge、所属分类显示 | 左侧面板 + 表格 |
| 加载 | `v-loading` | 表格加载 | 主区域 |

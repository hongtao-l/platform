## 一、业务背景

全能AI 是摄像头的智能看护管家功能，帮助用户让摄像头"看懂"画面，主动推送关注事件。采用单一套餐方案（¥9.9/月 全能AI），设备维度激活。用户开通后可在已激活设备上使用 AI消息、AI日报、AI搜索、关注事件管理等功能。

本需求文档描述 APP 端在「全能AI」模块中需要实现的功能，包括落地页展示、套餐购买、设备级激活状态管理、以及各子功能（AI搜索、AI日报、AI消息）的设备联动。

---

## 二、设备激活状态

开通全能AI服务后，每台设备独立控制是否启用 AI 功能。

| 设备状态 | 含义 |
|------|------|
| 已激活 | 该设备已启用 AI 功能，可使用 AI消息、AI日报、AI搜索等 |
| 未激活 | 该设备未启用 AI 功能，引导用户开通 |

**规则**：
- 仅已激活的设备可使用 AI 功能
- 未激活设备在各 AI 功能页面显示内联开通引导
- 页面默认选中第一台已激活的设备

## 三、入口

| 入口 | 触发方式 | 跳转目标 |
|------|------|------|

## 四、全能AI 落地页

### 4.1 未开通态 → 服务介绍开通页

页面由上至下分为 5 个区域：

**① 顶部 Banner**
- 全能AI 图标 + 主标题"全能AI · 智能看护管家" + 副标题
- 功能卡片：AI消息、AI日报、AI搜索 三个卡片

**② 套餐选择（左右并排，两个套餐）**
- 卡片竖版布局：套餐名称 → 价格 → 标签 → 四项权益打勾
- 套餐A：全能AI套餐 ¥9.9/月（连续订阅）
  - 标签"连续订阅"
  - 权益：AI消息推送、AI日报、AI搜索、关注事件管理
- 套餐B：全能AI套餐 ¥99/年（买断制）
  - 标签"买断制"
  - 权益：同套餐A
- 选中连续订阅后，下方显示提示文字："连续订阅套餐，到期自动按9.9元/月自动续费，可随时取消"

**③ 支付方式选择**
- 微信支付
- 支付宝

**④ 服务介绍**
- AI消息推送：摄像头自动识别画面中的人、宠物、车辆等，只推送你关注的事件
- AI日报总结：每日自动生成图文报告，汇总关注事件的完整时间线和一句话总结
- AI搜索：用自然语言搜索事件记录，快速定位关键片段
- 关注事件管理：自定义关注的人和事物，设置关键词，AI精准识别并优先推送

**⑤ 底部固定栏**
- "立即开通"按钮，按钮上显示对应套餐价格（¥9.9 或 ¥99）
- 按钮上方：协议勾选区（整行可点击切换）
  - 方形复选框 + 协议文字
  - 未勾选：灰边框空心框；已勾选：蓝底白对勾
  - 选中连续订阅时 → "已阅读并同意《云服务协议（含自动续费协议）》"
  - 选中买断制时 → "已阅读并同意《云服务协议》"
- 未勾选协议点击立即开通 → 弹窗提示"请先阅读并同意云服务协议"
  - 按钮：取消 / 同意（点击同意自动勾选协议，进入购买流程）

### 4.2 已开通态 → AI配置页

展示内容：
- 状态卡片：显示"全能AI" + 有效期 + "✓ 守护中"
- 关注事件管理区：
  - 智能推送开关（默认开启）
  - 提示文字"开启后仅推送关注事件消息"
  - 默认关注事件（4个）：
    - 儿童守护（小孩哭喊、跌倒、出门）
    - 包裹送达（快递或者外卖送达）
    - 厨房守护（厨房异常烟雾、明火）
    - 宠物守护（宠物乱拉乱尿、吃饭、喝水）
  - 事件列表（名称 + 描述 + 启停开关 + 编辑 + 删除）
  - "添加"按钮（显示当前数量/上限，如"添加 4/10"）→ 弹窗输入关注名称和事件描述
  - 最多添加10个关注事件，达到上限后弹 toast 提示"最多添加10个关注事件"

### 4.3 支付弹窗

- 金额：随选中套餐变化（¥9.9/月 或 ¥99/年）
- 支付方式：微信支付、支付宝
- 确认支付按钮

## 五、AI搜索 页面

### 5.1 页面结构

- 设备筛选栏（始终可见）：显示所有设备，默认选中当前已激活设备
- 搜索输入区（始终可见）：自然语言搜索框 + 搜索按钮 + 示例提示
- 内容区：
  - 设备已激活 → 猜你想搜 + 搜索结果列表
  - 设备未激活 → 内联引导提示（icon + "AI搜索" + 描述 + "立即开通全能AI"按钮）

### 5.2 设备切换行为

- 切换到已激活设备 → 正常显示搜索功能
- 切换到未激活设备 → 保留设备筛选栏和搜索输入区，内容区替换为内联引导提示
- **不弹 toast，不全屏锁死**

---

## 六、AI日报 页面

### 6.1 页面结构

- 日期栏（始终可见）：左右箭头切换日期 + 点击日期弹出日期选择器
- 设备筛选栏（始终可见）：显示所有设备，默认选中当前已激活设备
- 内容区：
  - 设备已激活 → 事件汇总卡片 + AI总结 + 底部生成按钮
  - 设备未激活 → 内联引导提示（icon + "AI日报" + 描述 + "立即开通全能AI"按钮）

### 6.2 设备切换行为

- 切换到已激活设备 → 正常显示日报内容
- 切换到未激活设备 → 保留日期栏和设备筛选栏，内容区替换为内联引导提示
- **未激活设备时右上角不显示历史日报入口**

---

## 七、消息页 联动

### 7.1 筛选

- 设备筛选：单选，无"全部设备"选项
- 日期筛选：单日，默认显示当前日期

### 7.2 事件筛选标签

标签根据当前设备是否开通全能AI，显示不同内容：

**已开通设备**（关注事件在前，基础事件在后）：
1. 全部
2. 老人看护（关注事件）
3. 宠物活动（关注事件）
4. 人员经过（关注事件）
5. 其他

**未开通设备**（仅基础事件）：
1. 全部
2. 运动侦测
3. 人形侦测

**交互规则**：
- 切换设备时自动重置为"全部"
- 标签栏支持鼠标滚轮横向滚动

### 7.3 "+关注"按钮

- 位置：事件筛选栏目右侧固定显示
- 行为：
  - 当前设备已激活 → 跳转全能AI配置页
  - 当前设备未激活 → 跳转服务介绍开通页

---

## 八、首页设备卡片 AI 入口

- 已开通设备：显示绿色文字"AI守护中"，点击跳转全能AI配置页
- 未开通设备：显示渐变标签"AI"，点击跳转服务介绍开通页

## 业务背景

套餐运营模块旨在为运营人员提供精细化的套餐推送管理能力。通过配置套餐运营位、套餐池和运营策略，结合用户分群，实现对不同用户群体的差异化套餐推送，提升套餐转化率和用户付费意愿。

---

## 名词解释

| 名词 | 解释 |
|---|---|
| 套餐运营位 | APP 内预留的套餐推送展示区域，用于触达用户并推荐套餐 |
| 运营位编码 | 套餐运营位的唯一标识，不可重复 |
| 套餐池 | 与运营位绑定的套餐集合，从 APP 下已创建的套餐中选择 |
| 套餐运营策略 | 针对一组运营位的套餐推送规则集合，包含静态指定和动态匹配两种模式 |
| 静态指定 | 手动指定各号位展示的套餐及运营标签 |
| 动态匹配 | 根据触发场景和匹配策略自动推荐套餐 |
| 套餐用户分群 | 可绑定套餐运营策略的用户集合，实现差异化套餐推送 |
| 冷却期 | 两次弹窗之间的最小间隔时间（小时） |
| 每日弹窗上限 | 单用户每天最多触发弹窗的次数 |

## 业务流程图

### 套餐运营配置流程

```
创建/选择 APP
    ↓
进入应用详情页
    ↓
【套餐运营位 Tab】管理运营位
    ↓
添加运营位（填写名称、编码）
    ↓
进入套餐池管理
    ↓
从 APP 套餐列表中选择套餐添加至套餐池
    ↓
返回应用详情页
    ↓
【套餐运营策略 Tab】管理运营策略
    ↓
添加/编辑套餐运营策略
    ├── 填写策略名称
    ├── 配置运营位
    │   ├── 静态指定：配置各号位套餐、运营标签、文案、冷却期
    │   └── 动态匹配：配置触发场景、匹配策略、匹配套餐类型
    ├── 配置全局策略（每日弹窗上限）
    └── 保存
```

### 套餐用户分群绑定策略流程

```
进入【套餐用户分群】
    ↓
查看分群列表
    ↓
点击【绑定策略】
    ↓
选择 APP（多选）
    ↓
选择对应 APP 下的套餐运营策略（单选）
    ↓
查看策略详细内容确认
    ↓
保存绑定
```

## UI 交互规范

### 表单验证

**必填项标记：**
- 所有必填项在标签后显示红色星号 `*`

**错误提示：**
- 提交时校验所有必填项
- 未填写的必填项在字段下方显示红色错误提示："此项为必填项"
- 对应字段边框变红
- 用户填写完整后，错误提示自动消失，字段边框恢复正常

**保存按钮状态：**
- 有必填项未填写时，保存按钮置灰不可点击
- 所有必填项填写完整后，保存按钮恢复可用

### 删除操作

**删除限制提示：**
- 当数据被其他模块引用时，无法删除
- 若被引用，弹出"无法删除"提示弹窗，说明原因和解决方案

**删除确认：**
- 可以删除的数据，点击删除按钮时显示确认对话框，用户确认后执行删除

### 侧边抽屉

**宽度规范：**
- 标准抽屉：480px（简单表单）
- 宽抽屉：600px（中等复杂度表单）
- 超宽抽屉：640px（复杂表单，如套餐运营策略配置）

### 列表添加/编辑弹窗规则

所有列表的添加弹窗和编辑弹窗内容完全一致，唯一区别是弹窗标题文案：
- 添加：**添加 XX**
- 编辑：**编辑 XX**

## 详细功能说明

### 1. 应用管理 — 套餐运营位管理 Tab

**位置：** 应用编辑页新增 Tab【套餐运营位】

**列表字段：**

| 字段 | 说明 |
|---|---|
| 运营位名称 | 运营位的展示名称 |
| 运营位编码 | 运营位唯一标识，不可重复 |
| 创建时间 | 运营位创建时间 |
| 操作 | 套餐池、编辑、删除 |

**操作说明：**

- 套餐池：点击跳转至套餐池管理页面（三级页）
- 编辑：打开侧边抽屉编辑运营位信息
- 删除：检查是否被运营策略引用，被引用时不可删除

**删除限制：**
- 运营位被套餐运营策略引用时，无法删除
- 弹出提示："该运营位已被套餐运营策略引用，无法删除。请先在套餐运营策略中移除该运营位。"

**添加/编辑套餐运营位（侧边抽屉，宽度 480px）：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 运营位名称 | 文本输入 | 是 | 运营位展示名称 |
| 运营位编码 | 文本输入 | 是 | 唯一标识，不可重复 |

**校验规则：**
- 运营位名称、运营位编码均为必填
- 运营位编码不可与已有编码重复，重复时显示："该编码已存在，请重新输入"

---

### 2. 应用管理 — 套餐池管理（三级页）

**页面路径：** 应用管理 / {APP名称} / {运营位名称} · 套餐池

**页面结构：**

顶部展示面包屑导航及返回按钮，下方为套餐池列表。

**列表字段：**

| 字段 | 说明 |
|---|---|
| 序号 | 列表序号 |
| 套餐代码 | 套餐唯一标识 |
| 套餐名称 | 套餐展示名称 |
| 套餐价格 | 套餐售价 |
| 套餐类型 | 云存储 / AI服务 |
| 订阅类型 | 是 / 否 |
| 操作 | 移除 |

**操作说明：**

- 添加套餐：点击【添加套餐】按钮，弹出侧边抽屉，展示该 APP 下所有已创建的套餐列表（含套餐代码、套餐名称、套餐价格、套餐类型、订阅类型），支持多选后批量添加
- 移除：点击可将该套餐从套餐池中移除，移除前显示确认对话框

**添加套餐（侧边抽屉，宽度 600px）：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 套餐列表 | 多选复选框 | 是 | 展示 APP 下所有套餐，支持多选 |

- 已在套餐池中的套餐显示为已选中且置灰，不可重复添加
- 支持按套餐名称/代码模糊搜索

---

### 3. 应用管理 — 套餐运营策略管理 Tab

**位置：** 应用编辑页新增 Tab【套餐运营策略】

**列表字段：**

| 字段 | 说明 |
|---|---|
| 策略名称 | 策略展示名称 |
| 包含运营位 | 该策略配置的运营位数量 |
| 全局策略 | 展示每日弹窗上限配置值 |
| 操作 | 编辑、删除 |

**删除限制：**
- 套餐运营策略被用户分群绑定时，无法删除
- 弹出提示："该套餐运营策略已被用户分群引用，无法删除。请先在套餐用户分群中移除该策略的绑定。"

**添加/编辑套餐运营策略（侧边抽屉，宽度 640px）：**

**基础信息：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 策略名称 | 文本输入 | 是 | 策略展示名称 |

**运营位配置（可动态添加/删除多个运营位，不可重复选择同一运营位）：**

每个运营位配置块包含：

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 选择运营位 | 下拉选择 | 是 | 从当前 APP 运营位中选择，不可重复 |
| 套餐策略 | 单选 | 是 | 静态指定 / 动态匹配 |

**套餐策略 — 静态指定：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 1号位套餐 | 下拉选择（模糊查询） | 是 | 从套餐池中选择套餐 |
| 1号位运营标签 | 下拉选择 | 否 | 折扣 / 新用户 / 免费试用 |
| 1号位文案 | 文本输入 | 否 | 自定义展示文案 |
| 2号位套餐 | 下拉选择（模糊查询） | 否 | 从套餐池中选择套餐 |
| 2号位运营标签 | 下拉选择 | 否 | 折扣 / 新用户 / 免费试用 |
| 2号位文案 | 文本输入 | 否 | 自定义展示文案 |
| 3号位套餐 | 下拉选择（模糊查询） | 否 | 从套餐池中选择套餐 |
| 3号位运营标签 | 下拉选择 | 否 | 折扣 / 新用户 / 免费试用 |
| 3号位文案 | 文本输入 | 否 | 自定义展示文案 |
| 冷却期（小时） | 数字输入 | 是 | 两次弹窗的最小间隔，字段下方显示小字说明："两次弹窗的最小间隔" |

**套餐策略 — 动态匹配：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 触发场景 | 下拉选择 | 是 | 云存循环覆盖 / SD卡循环覆盖 |
| 匹配策略 | 下拉选择 | 是 | 步进升级 / 最优折扣 / 历史偏好 |
| 匹配套餐类型 | 下拉选择 | 是 | 云存储 / AI |

**全局策略配置：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 每日弹窗上限（次/天） | 数字输入 | 是 | 单用户每天最多触发弹窗次数 |

**校验规则：**
- 策略名称、至少一个运营位配置、全局策略均为必填
- 静态指定时：1号位套餐、冷却期为必填
- 动态匹配时：触发场景、匹配策略、匹配套餐类型均为必填
- 同一策略内，同一运营位只能配置一次；重复时下拉框标红并显示："该运营位已被添加，请勿重复配置"

### 4. 用户管理 — 套餐用户分群

**页面路径：** 用户管理 / 套餐用户分群

**设计说明：**
用户分群由数据仓库同步，运营人员可在此页面查看分群信息并为各分群绑定套餐运营策略，实现差异化套餐推送。

**列表字段：**

| 字段 | 说明 |
|---|---|
| 序号 | 列表序号 |
| 分群名称 | 分群展示名称（由数据仓库同步） |
| 分群编码 | 分群唯一标识（由数据仓库同步） |
| 用户数量 | 当前分群内的用户数（由数据仓库同步） |
| 运营策略 | 已绑定时显示策略名称+【查看】+【绑定策略】；未绑定时显示"未绑定"+【绑定策略】 |

**查看绑定策略（侧边抽屉，宽度 600px）：**

点击【查看】按钮弹出，展示：
- 当前分群绑定的套餐运营策略详细内容（仅查看）
- 对于已绑定策略的 APP，展示：
  - 策略名称、全局策略（每日弹窗上限）
  - 运营位配置明细（运营位名称、套餐策略类型、配置详情）
- 对于未绑定的 APP，显示 APP 信息和"未绑定"状态
- 底部仅有【关闭】按钮

**绑定策略（侧边抽屉，宽度 600px）：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 选择 APP | 多选复选框 | 是 | 勾选需要配置套餐运营策略的 APP，支持全选 |
| 套餐运营策略 | 下拉选择（单选） | 条件必填 | 勾选 APP 后必须选择对应策略；选择后展示策略内容预览（策略名称、每日弹窗上限、运营位配置明细） |

**校验规则：**
- 至少勾选一个 APP
- 勾选 APP 后必须选择对应的套餐运营策略

---

### 删除限制规则（全局）

**套餐运营位删除限制：**
- 运营位被套餐运营策略引用时，无法删除
- 弹出提示："该运营位已被套餐运营策略引用，无法删除。请先在套餐运营策略中移除该运营位。"

**套餐运营策略删除限制：**
- 策略被用户分群绑定时，无法删除
- 弹出提示："该套餐运营策略已被用户分群引用，无法删除。请先在套餐用户分群中移除该策略的绑定。"

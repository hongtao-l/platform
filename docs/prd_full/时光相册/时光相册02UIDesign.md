# 时光相册 — 设计说明

> v1.1：设备设置页入口，4 分类 Tab 图片库，支持日期筛选和生成视频（不限张数、历史日期限1次、每日仅保留最新）

## 页面结构

本需求在 CareCam Pro APP 中新增时光相册模块：

```
CareCam Pro
├── 设备设置页（改造）
│   └── 设置列表新增「时光相册」项 → 进入时光相册主页
│
└── 时光相册（新建页面）
    ├── 顶部返回栏
    ├── 分类 Tab 栏（常规 / 宠物 / 老人 / 儿童）
    ├── 日期筛选栏
    ├── 图片网格区（按日期分组，瀑布流）
    ├── 底部操作栏（生成视频 + 视频列表入口）
    │
    ├── 图片选择模式（底部弹出层）
    │   ├── 勾选图片（不限张数）
    │   ├── 已选计数 + 预览条
    │   └── 校验（当日次数 + 历史日期）→ 确认生成 → 等待动画 → 成功/失败提示（同日旧视频替换）
    │
    ├── 视频列表页（子页面）
    │   ├── 视频卡片列表
    │   └── 每个卡片: 封面 + 生成时间 + 有效期倒计时 + 操作
    │
    └── 视频播放页（全屏）
        └── 视频播放器 + 下载/分享按钮
```

---

## 布局详述

### 区域0: 设备设置页（改造）

**改动**：在 `settingsItems` 数组中追加一项：

| 属性 | 值 |
|------|------|
| title | 时光相册 |
| desc | 智能分类、生成视频 |
| icon | photo-o |
| iconColor | #E91E63 |
| bgColor | #FCE4EC |
| action | time_album |

**交互**：点击 → `router.push({ path: '/time-album', query: { deviceId, deviceName } })`

---

### 区域1: 时光相册主页

#### 区域1.1: 顶部返回栏

- 左侧：`van-icon name="arrow-left"` + 标题「时光相册」
- 右侧：`van-icon name="info-o"` → 点击弹窗显示功能说明
- 背景白色，底部 1px 分割线

#### 区域1.2: 分类 Tab 栏

- **组件**：自定义 Tab 栏（非 van-tabs），风格与设备首页家庭分组标签一致
- **内容**：4 个 chip 按钮横向排列：「常规」「宠物」「老人」「儿童」
- **样式**：
  - 默认：透明底，灰色文字，`font-size: 13px`
  - 选中：`$primary-color` 底，白色文字，`border-radius: $radius-pill`
  - 容器：水平滚动（`overflow-x: auto`），左右 16px padding，间距 8px
- **行为**：点击切换分类，触发对应分类图片加载

#### 区域1.3: 日期筛选栏

- **位置**：分类 Tab 栏下方
- **组成**：
  - 日期范围显示：「2026/06/15 - 2026/06/22」
  - 右侧筛选图标 `van-icon name="filter-o"`
- **交互**：点击 → 底部弹出 `van-popup`，内含 `van-date-picker`（范围选择模式），确认后更新筛选 + 刷新图片库

#### 区域1.4: 图片网格区

- **组件**：CSS Grid 三列等宽网格，`gap: 4px`，`padding: 12px 16px`
- **布局**：按日期分组，每组顶部显示日期标签
  - 「今天」「昨天」「本周」「2026/06/15」等
  - 标签样式：`font-size: 12px`，灰色，左对齐，`margin: 8px 0 4px`
- **图片卡片**：
  - 1:1 正方形缩略图，`border-radius: $radius-sm`
  - 右下角叠拍时间 HH:mm（半透明黑底白字，`font-size: 10px`）
  - 使用 `van-image` 懒加载，`fit="cover"`，含加载失败占位图
- **滚动**：区域可垂直滚动，触底自动加载更多（每页 20 张）
- **空态**：
  - 单分类无图片：居中显示 `van-empty`，描述「暂无该类图片」
  - 全部无图片：居中显示「暂无设备抓拍图片」，插图使用默认
- **加载态**：首次加载显示 `van-loading` 居中；加载更多时底部显示 `van-loading size="small"`

#### 区域1.5: 底部操作栏

- **位置**：页面底部固定，白色背景，顶部分割线
- **布局**：flex 横排，左右 16px padding，`padding: 12px 16px`
- **组成**：
  - **生成视频按钮**：`van-button type="primary" block`（占满宽度）
    - 可用态：蓝色实心，「生成视频」文字 + `van-icon name="video-o"`
    - 禁用态（今日已用 2 次）：灰色，文字「今日已用完」，不可点击
    - 半禁用态（今日已用 1 次）：蓝色实心，「生成视频（剩余 1 次）」
  - **视频列表入口**：`van-icon name="tv-o"` 徽标 + 文字，放在按钮上方或右侧
    - 如有视频：显示红色圆点 badge
    - 点击 → 进入视频列表页

---

### 区域2: 图片选择模式

#### 区域2.1: 选择模式激活

- **触发**：点击主页底部「生成视频」按钮
- 校验当日生成次数和历史日期限制，通过后进入选择模式

#### 区域2.2: 选择模式界面

- **顶部提示栏**：
  - 关闭按钮（×）+ 提示文字「选择图片生成视频」
  - 背景 `$primary-light`
- **图片网格**：
  - 每张图片右上角出现 `van-checkbox` 圆形选择框
  - 选中的图片：边框高亮 `2px $primary-color`，checkbox 勾选
  - 未选中图片：半透明遮罩（opacity 0.6）
  - 不限制选择张数
- **底部操作栏**：
  - 左侧：已选 N 张 + 所选图片水平滚动预览条（30px 小缩略图）
  - 右侧：「开始生成」按钮 → `van-button type="primary"`
- **行为**：
  - 点击图片 → 勾选/取消
  - 点击「开始生成」→ 弹确认弹窗 → 确认后进入生成等待

#### 区域2.3: 生成等待

- **全屏遮罩**：`van-overlay` + 中央内容
  - 旋转动画 `van-loading type="spinner" size="48px"`
  - 文字：「视频生成中...」
  - 进度条（如有上传进度）或百分比
- **超时 60s 后**：自动隐藏，Toast「视频生成超时」
- **成功后**：隐藏 overlay，Toast「视频生成成功」，跳转视频列表
- **失败后**：隐藏 overlay，Toast「视频生成失败，请重试」

---

### 区域3: 视频列表页

#### 区域3.1: 页面结构

- **顶部返回栏**：返回按钮 + 标题「我的视频」
- **内容**：
  - 有视频 → 视频卡片列表，每项包含：
    - 封面缩略图（16:9），`van-image fit="cover"`
    - 右下角视频时长标签（mm:ss）
    - 标题行：「时光视频」+ 生成日期
    - 有效期倒计时文案：「剩余 28 天」（≤3 天时红色警示）
    - 操作行：播放 ▶ / 下载 ⇩ / 分享 ↗（`van-icon` + 文字）
  - 无视频 → `van-empty` 居中，描述「暂无生成的视频」，插图
- **列表间距**：卡片间距 12px，左右 16px padding
- **视频有效期**：过期的视频卡片灰色半透明，不可播放，显示「已过期」标签

---

### 区域4: 视频播放页

- **全屏播放器**：`<video>` 元素，controls 属性
- **顶部**：返回按钮（半透明浮层），点击回到视频列表
- **底部**：下载按钮 + 分享按钮（半透明浮层）
  - 下载：保存到系统相册
  - 分享：调起系统分享面板

## 状态说明

| 状态 | 触发条件 | 展示内容 |
|------|------|------|
| 正常态 | 图片数据就绪 | 分类 Tab + 日期筛选 + 图片网格 + 底部操作栏 |
| 空态 — 全部 | 所有分类均无图片 | 中间 `van-empty`「暂无设备抓拍图片」 |
| 空态 — 单分类 | 某分类无图片 | 切换 Tab 后显示「暂无该类图片」 |
| 空态 — 视频 | 从未生成视频 | 视频列表页显示「暂无生成的视频」 |
| 加载态 — 首次 | 首次进入页面 | 居中 `van-loading` |
| 加载态 — 更多 | 滚动到底部 | 底部 `van-loading size="small"` |
| 加载态 — 生成中 | 正在合成视频 | 全屏遮罩 + spinner + 「视频生成中...」 |
| 边界态 — 次数用完 | 当日已生成 2 次 | 底部按钮置灰「今日已用完」 |
| 边界态 — 次数剩余 1 | 当日已生成 1 次 | 按钮显示「生成视频（剩余 1 次）」 |
| 边界态 — 历史日期不可生成 | 该日期已生成过视频 | 选择该日期时按钮置灰，Toast「该日期已生成过视频」 |
| 边界态 — 视频临近过期 | 剩余 ≤ 3 天 | 倒计时文案红色警示 |
| 边界态 — 视频已过期 | 超过 30 天 | 卡片灰色半透明 +「已过期」标签 |
| 错误态 — 网络 | 加载/生成时断网 | Toast「网络异常，请检查网络」 |
| 错误态 — 服务端 | 接口返回错误 | Toast「加载失败，请重试」+ 重试可点击 |
| 错误态 — 生成超时 | 合成 > 60s | Toast「视频生成超时」，不消耗次数 |
| 错误态 — 图片失效 | 单张缩略图加载失败 | 显示默认灰色占位图 |

## 组件清单

| 组件 | 用途 | 所在区域 |
|------|------|------|
| van-icon | 图标（返回/信息/筛选/视频/下载/分享） | 全局 |
| van-image | 图片缩略图懒加载 | 图片网格、选择模式预览、视频封面 |
| van-button | 生成视频/确认/下载/分享按钮 | 底部操作栏、视频列表、视频播放 |
| van-loading | 加载指示器 | 首次加载、滚动加载更多、生成等待 |
| van-empty | 空态占位 | 无图片、无视频 |
| van-popup | 日期筛选弹出面板 | 日期筛选栏 |
| van-date-picker | 日期范围选择 | 日期筛选 popup 内 |
| van-checkbox | 图片多选勾选 | 图片选择模式 |
| van-overlay | 遮罩层 | 图片选择模式背景、生成等待遮罩 |
| van-dialog | 确认弹窗 | 确认开始生成视频 |
| van-swipe | 图片详情左右滑动 | 大图浏览（如实现） |

# 活动位运营 — UI设计

## 1. 设计概述

### 1.1 设计依据
基于 [商城位运营 UI 模式](../../../project/design/platform_pkg/ops-mall.html)，复用其策略列表 + 步骤式抽屉 + 套餐穿梭框的成熟交互模式，新增活动周期、活动位置选择、活动媒资上传、活动链接配置、活动状态流转等差异化功能。

### 1.2 设计目标
- 与商城位运营保持一致的视觉风格和交互习惯
- 活动特有字段清晰突出，不增加认知负担
- 状态流转可视化
- 媒资上传与位置联动校验

## 5. 交互说明

### 5.1 提交策略
1. 3步表单全部填写完毕，点击"提交"/"发布"
2. 前端校验：必填项、日期合法性、链接格式、媒资是否上传
3. 校验通过 → 策略创建，状态设为"草稿"；点击"发布" → 状态变为"进行中"
4. 关闭抽屉，列表刷新

### 5.2 位置切换
- 切换位置时，Step 3 的媒资规格提示联动更新
- 已上传的媒资若尺寸不符新位置则提示重新上传

### 5.3 弹窗频率
- 仅在位置为"弹窗"时显示
- 默认选中"每天一次"

### 5.4 状态操作
- 列表操作列动态渲染按钮
- 草稿策略 → 可见"发布"按钮，点击后状态变"进行中"
- 进行中策略 → 可"暂停"，暂停后可"恢复"或"结束"
- 已结束/已过期策略 → 可"复制新建"

### 5.5 活动到期处理
- 前端定时检查，到达 endTime 自动将"进行中"改为"已过期"
- 列表样式置灰

### 8.3 ActivityBanner — 首页轮播

**组件定位**：展示 `slot_banner` 位置的有效活动，支持多活动轮播。

**UI 规格**：
```
┌─────────────────────────────────┐
│ ┌─────────────────────────────┐ │
│ │  [Banner 图片 750×400]       │ │  ← van-swipe 轮播
│ │   618大促Banner活动           │ │    aspect-ratio: 343/120
│ │                              │ │    border-radius: 12px
│ │   ● ● ●  (指示器)            │ │    overflow: hidden
│ └─────────────────────────────┘ │
└─────────────────────────────────┘
  margin: 12px 16px 0
  autoplay: 3000ms
```

| 属性 | 值 |
|------|------|
| 容器样式 | `margin: 12px 16px 0; border-radius: 12px; overflow: hidden` |
| 图片比例 | `aspect-ratio: 343 / 120` |
| 轮播间隔 | 3000ms |
| 指示器 | `van-swipe` indicator-dots |
| 点击行为 | `router.push(url)` 或 `window.open(url)` |
| 空状态 | `v-if="store.activeBanners.length > 0"` 不渲染 |
| 加载失败 | 渐变占位 + 活动标题文字 |
| 排序 | 按 `priority` 降序 |

**状态定义**：
- **有活动**：渲染 van-swipe 轮播区域
- **无活动**：区域不渲染（`v-if`），下方设备列表自然上移
- **图片加载失败**：`@error` 隐藏 img，显示渐变背景 + 活动名称文字

### 8.4 ActivityPopup — 启动弹窗

**组件定位**：展示 `slot_popup` 位置的有效活动，支持频率控制。

**UI 规格**：
```
┌───────────────────────┐
│                       │
│  ┌─────────────────┐  │
│  │                  │  │
│  │   [媒资图片]      │  │  ← 300×200 比例
│  │   600×800        │  │    渐变背景兜底
│  │                  │  │    图片失败→显示 🎁 占位
│  │                  │  │
│  └─────────────────┘  │
│                       │
│   618大促Banner活动     │  ← 活动名称 title
│                       │
│  ┌─────────────────┐  │
│  │    立即查看       │  │  ← van-button round block primary
│  └─────────────────┘  │
│       暂不参与         │  ← 灰色文字链接
│                       │
│                    ✕  │  ← 右上角关闭
└───────────────────────┘
  van-popup: position="center", round, width: 300px
```

| 属性 | 值 |
|------|------|
| 弹窗宽度 | 300px |
| 图片比例 | `aspect-ratio: 300/200` |
| 弹出延迟 | 500ms |
| "立即查看" | 跳转链接 + 关闭弹窗 + 记录 dismiss |
| "暂不参与" | 关闭弹窗 + 记录 dismiss |
| 关闭按钮 | 同"暂不参与" |

**频率控制**：
| 频率 | 行为 |
|------|------|
| `always` | 每次启动都弹 |
| `once_daily` | 当天未 dismiss 才弹（localStorage 存 YYYY-MM-DD） |
| `once_weekly` | 本周未 dismiss 才弹（localStorage 存周一日期） |

**状态定义**：
- **有弹窗活动 + 未 dismissed**：延迟 500ms 弹出
- **同一会话已弹过**：`sessionDismissed` flag，不重复弹
- **已 dismiss（当日/当周）**：不弹
- **无有效弹窗活动**：不弹

### 8.5 ActivityFab — 悬浮按钮

**组件定位**：展示 `slot_floating` 位置优先级最高的活动。

**UI 规格**：
```
┌──────────────────────────┐
│                          │
│                          │
│                   ┌──┐   │
│                   │✕ │   │  ← 右上角关闭按钮（12px）
│                   │  │   │    圆形，白色背景 + 阴影
│                   └──┘   │
│                   52×52  │  ← 圆形，图片填充
│                          │    无图片时默认图标
│                     ↓    │
│                 80px     │
│                          │
└──────────────────────────┘
    16px →|←
  position: fixed, bottom: 80px, right: 16px, z-index: 100
```

| 属性 | 值 |
|------|------|
| 尺寸 | 52×52px 圆形 |
| 定位 | `fixed`, `bottom: 80px`, `right: 16px`, `z-index: 100` |
| 关闭按钮 | 右上角 12px × 按钮，点击当天隐藏 |
| 点击行为 | 跳转 `activityUrl` |
| 动效 | pulse 动画（周期性缩放吸引注意） |
| 空状态 | `v-if="store.activeFab"` 不渲染 |

**状态定义**：
- **有悬浮活动**：渲染圆形按钮 + pulse 动画
- **关闭当天**：localStorage 记录日期，当天不再显示
- **无有效活动**：不渲染

### 8.7 交互流程总览

```
首页加载
  │
  ├─→ ActivityBanner: watch activeBanners
  │     └─→ 有数据 → 渲染轮播 → 点击跳转
  │
  ├─→ ActivityPopup: watch activePopups + forcePopup
  │     └─→ 有数据 + 未dismiss → 延迟500ms → 弹窗
  │           ├─→ "立即查看" → 跳转 + dismiss
  │           └─→ "暂不参与" → dismiss
  │
  ├─→ ActivityFab: watch activeFab
  │     └─→ 有数据 → 渲染悬浮按钮
  │           ├─→ 点击图片 → 跳转
  │           └─→ 点击关闭 → 当天隐藏
  │
  └─→ ActivityDemoPanel: 手动操作
        ├─→ 添加演示 → store.addDemo(position)
        ├─→ 移除演示 → store.removeDemo(position)
        ├─→ 触发弹窗 → store.triggerPopupNow()
        └─→ 重置 → store.resetAll() + initMockData()
```

## 3. 交互行为

### 3.1 模板面板展开/收起（手风琴）

```
点击某层的「▸ 模板名」行
  ├── 当前面板收起 → 展开该层模板卡片，箭头 ▸ → ▾
  ├── 当前面板展开 → 收起，箭头 ▾ → ▸
  └── 另一层面板已展开 → 先收起另一层面板，再展开当前面板（同时最多展开一个）
```

| 行为 | 触发 | 效果 |
|------|------|------|
| 展开 | 点击某层收起态的模板选择行 | 下方滑出模板卡片面板 |
| 收起 | 点击当前展开层的模板选择行 | 面板收起 |
| 切换 | 点击另一层的模板选择行 | 当前层自动收起，新层展开 |
| 选中模板 | 点击面板内模板卡片 | 更新该层选中模板，✓ 标记移动 |
| 横向滚动 | 鼠标滚轮在卡片区域 | 仅横向滚动，不触发外层竖向滚动 |

### 3.2 管理模板

「管理模板」按钮出现在**展开的面板底部**，始终可见。点击打开管理模板弹窗。

## 4. 模板卡片

### 4.1 卡片规格

```
┌────────────┐
│            │
│            │
│  [缩略图]   │  ← 9:16 手机比例
│            │   aspect-ratio: 9/16
│            │
│            │
│  模板名称   │  ← 13px，居中
│  ✓ 已选    │  ← 右上角蓝色圆形 badge
└────────────┘
  160px 宽
```

| 属性 | 值 |
|------|------|
| 宽度 | 160px（`min-width: 160px`，`flex-shrink: 0`） |
| 缩略图 | aspect-ratio 9:16，圆角 8px，`object-fit cover` |
| 名称 | 13px，居中，`var(--text-primary)` |
| 选中态 | 蓝色边框 2px `var(--primary-color)` + 右上角蓝色圆形 ✓ badge |
| 未选中态 | 1px `var(--border-light)` |
| hover | 边框变蓝，cursor pointer |
| 排列 | flex，`overflow-x: auto`，gap 10px |
| 滚动 | 横向滚动条 4px + 鼠标滚轮横向滚动（`@wheel.prevent`） |

### 4.2 无独立预览区

卡片本身 9:16 比例足够大，不再需要下方单独预览区。

## 7. 状态一览

| 状态 | 套餐详情 | 层级模板行 | 模板面板 |
|------|------|------|------|
| 未选漏斗 | 隐藏 | 隐藏 | 隐藏 |
| 已选漏斗 + 面板收起 | 展示 | 展示（▸） | 隐藏 |
| 已选漏斗 + 某层面板展开 | 展示 | 该层 ▾ 其他 ▸ | 该层展示 |
| 漏斗无套餐 | 「暂无套餐」 | 仍展示 | 仍可展开 |

## 8. 弹窗层级

```
策略抽屉 (el-drawer, z=2000)
  └── 管理模板弹窗 (el-dialog, z=9999)
        ├── 添加/编辑子弹窗 (el-dialog, z=19999)
        └── 模板图片预览弹窗 (el-dialog, z=29999, width=700px)
```

---

> 模板卡片使用 CSS `aspect-ratio: 9/16` + 横向 `overflow-x: auto`，支持鼠标滚轮横向滚动且不穿透到外层。

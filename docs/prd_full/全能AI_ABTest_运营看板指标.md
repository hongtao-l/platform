## 二、核心指标

### 2.1 实验概览指标（看板顶部卡片）

| 指标 | 计算公式 | 统计周期 | 可视化 |
|------|------|------|------|
| 总曝光 UV | COUNT(DISTINCT user_id) WHERE event_type='experiment_view' | 实验运行期间 | 数字卡片 |
| 总转化 UV | COUNT(DISTINCT user_id) WHERE event_type='pay_success' | 实验运行期间 | 数字卡片 |
| 整体转化率 | 总转化UV / 总曝光UV × 100 | 实验运行期间 | 百分比卡片 |
| 总收入 | SUM(price) WHERE event_type='pay_success' | 实验运行期间 | 金额卡片 |

### 2.2 变体对比指标（A/B 对比卡片）

| 指标 | 数据口径 | 说明 |
|------|------|------|
| 页面曝光 UV | 各变体 experiment_view 去重 user_id | 流量是否均匀 |
| 套餐选择率 | plan_select UV / experiment_view UV | 页面吸引力 |
| 支付点击率 | pay_click UV / experiment_view UV | 按钮吸引力 |
| 支付成功率 | pay_success UV / pay_click UV | 支付流程阻力 |
| 转化率 | pay_success UV / experiment_view UV | 最终转化效果 |
| 客单价 | SUM(price) / pay_success UV | 付费金额差异 |
| 总收入 | SUM(price) | 总营收对比 |

### 2.3 套餐维度指标

| 指标 | 数据口径 | 说明 |
|------|------|------|
| Basic 选择占比 | plan_type='basic' 的 plan_select 数 / 总 plan_select 数 | 低价套餐偏好 |
| Pro 选择占比 | plan_type='pro' 的 plan_select 数 / 总 plan_select 数 | 高价值套餐偏好 |
| Basic 转化率 | basic 的 pay_success / basic 的 plan_select | 低价套餐成交 |
| Pro 转化率 | pro 的 pay_success / pro 的 plan_select | 高价值套餐成交 |
| 升级率 | basic → pro 的升级事件 | 套餐升级意愿 |

### 2.4 分群维度指标

| 指标 | 说明 |
|------|------|
| 各分群实验数量 | 当前运行中/已结束的实验数 |
| 分群转化率均值 | 该分群下所有实验的平均转化率 |
| 分群收入贡献 | 该分群的总收入 |

## 四、数据校验规则

| 校验项 | 规则 | 异常处理 |
|------|------|------|
| 流量均匀性 | max(A曝光, B曝光) / min(A曝光, B曝光) < 1.2 | 超过1.2提示流量分配异常 |
| 重复事件 | 同一 user_id + event_type 1分钟内去重 | 标记重复数据 |
| 支付金额校验 | price ∈ {9.9, 19.9, 3.9} | 异常金额标记 |
| 数据延迟 | event 创建时间 > 当前时间 + 5min | 标记异常时间戳 |

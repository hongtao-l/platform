# 广告策略运营PRD
## 【应用管理】模块
    -应用列表：APP名称、APPID、应用平台、应用区域、收款方式、套餐底价、开放类型、收款账号、审核状态、操作
    -应用操作：
        -编辑：
            -点击跳转至二级页面
            -二级页面显示内容
                -APP基础信息
                -4个tab可切换
                    -版本管理
                    -Push消息推送
                    -收款账号管理
                    -广告位管理
                        -广告位列表：广告位名称、广告位ID、广告类型、创建时间、操作
                        -广告位操作：
                            -广告源配置：点击进入二级页面
                                1.返回：点击返回上级页面
                                2.广告源列表：广告标题、广告ID、广告平台、广告源ID
                                3.广告源操作：
                                    -编辑：侧边弹窗
                                        -广告标题：录入文本
                                        -广告平台：下拉选择（fengchuan）
                                        -广告源ID：录入文本
                    -广告策略管理
                        -广告策略列表：策略ID、策略名称、包含广告位（数量）、全局策略（显示配置项和值）
                        -广告策略操作：
                            -添加：
                                -点击弹出侧边弹窗
                                -策略名称：录入文本
                                -配置广告位
                                    -选择广告位
                                    -配置广告位控制参数
                                        -展示状态：开启/关闭，开启状态先需配置以下参数
                                        -频次上限（次/小时）
                                        -展示间隔（s）
                                        -刷新间隔（s）
                                    -配置广告源（下拉选择）
                                -配置全局策略
                                    -开屏/插屏冷却间隔（s）：录入文本
                            -编辑：
                                -同添加一致
                            -删除：
        -创建应用
        -删除应用

## 【用户管理】模块
### 【广告用户分群】子菜单
        -用户分群列表：序号、分群名称、分群编码、分群类型、用户数量、广告策略、创建时间
            -广告策略：显示策略名称+ID；加个查看按钮，可查看该策略中的具体配置信息
        -用户分群操作：
            -创建分群：
                -分群名称：录入文本；必填
                -分群级别：下拉文本（P0~P5）；必填
                -分群编码：固定前缀user_group_,后面是录入文本；必填，不可重复
                -分群描述：非必填项
                -分群规则：
                    -规则组管理：组内支持添加/删除规则，每个规则定义格式如下
                        -属性标签：下拉选择（设备型号、地理位置、绑定设备数、直播查看、回放点击、告警推送点击、分享设备、云存储状态、存储卡状态）
                        -操作符：下拉选择
                        -具体值：根据属性判断是录入还是其他类型
                        -时间窗口：选填
                    -规则组内可添加多个规则
                    -规则之间以逻辑连接符（或/且）连接，规则组之间以辑连接符（或/且）连接
                    -在同一层级内，所有的逻辑连接符（或/且）必须保持联动一致
                -广告策略
                    -勾选需要配置广告策略的APP，可多选
                    -配置勾选的APP的广告策略，单选
            -编辑
                -同创建分群一致，分群编码无法变更，弹窗标题变更为：编辑分群
            -删除
### 【免广告用户】子菜单
        -免广告用户列表：用户ID、用户账号、用户分群、有效期、结束时间、操作
        -免广告用户操作：
            -添加用户：
                -用户ID：录入文本
                -用户分群：选择文本
                -生效时长（天）：录入文本
            -删除

# 套餐运营PRD
## 【应用管理】模块
    -应用编辑页面
        -新增tab【套餐运营位】管理
            -套餐运营位列表：运营位名称、运营位编码、创建时间、操作
            -操作：
                -添加套餐运营位：
                    -运营位名称：必填
                    -运营位编码：必填，不可重复
                -套餐管理：点击进入【套餐管理】列表页面
                -套餐漏斗：点击进入【套餐漏斗】管理页面
                -编辑
                -删除
            -【套餐管理】页面
                -套餐池列表：序号、套餐ID、套餐名称、套餐价格、套餐类型（云存储、AI服务）、订阅类型（是、否）、操作
                -操作：
                    -添加套餐：侧边弹窗，填写添加字段
                        -套餐名称：录入文本
                        -推荐标题：录入文本
                        -推荐文案：录入文本
                        -选择套餐服务
                            -应用方向：单选
                                -设备套餐
                                    -生效时长：下拉选择（7天、15天、30天、1年）
                                    -套餐服务：下拉选择（7天事件云存、15天事件云存）
                                        -套餐服务详情：根据选择的套餐服务动态显示内容，未选择服务则不显示详情
                                            -套餐服务名称：7天事件云存
                                            -服务类型：事件云存
                                            -可用设备上限：1台
                                            -平台成本：￥8.00 / ＄1.20
                                -用户套餐
                                    -生效时长：下拉选择（3个月、6个月）
                                    -套餐服务：下拉选择（短信套餐、免广告-年）
                                        -套餐服务详情：根据选择的套餐服务动态显示内容，未选择服务则不显示详情
                                            -套餐服务名称：短信套餐
                                            -服务类型：短信套餐
                                            -平台成本：￥8.00 / ＄1.20
                        -内购key配置：非必填
                            -apple key：录入文本
                            -google key：录入文本
                        -订阅类型：单选
                            -非订阅
                                -划线价/原价（￥）：录入文本，数值，限制非负数
                                -划线价/原价（＄）：录入文本，数值，限制非负数
                                -售价（￥）：录入文本，数值，限制非负数
                                -售价（＄）：录入文本，数值，限制非负数
                            订阅制
                                -订阅文案：录入文本
                                -划线价/原价（￥）：录入文本，限制非负数
                                -划线价/原价（＄）：录入文本，限制非负数
                                -免费试用：开关
                                    -开启状态
                                        -试用周期（天）：录入文本
                                        -过滤条件：下拉筛选（从未试用过此套餐、从未试用过该服务类型、不限制）
                                    -关闭状态
                                        -空
                                -订阅售价（￥）：录入文本，限制非负数
                                -订阅售价（＄）：录入文本，限制非负数
                        -套餐标签：下拉选择（免费试用、限时优惠、新用户特惠）
                        -套餐描述标签：下拉选择（7天循环事件录制、7天消息回溯）
                        -确定：关闭弹窗，生成创建的套餐条目
                        -取消：关闭弹窗
                        -关闭：关闭弹窗
                    -编辑：
                    -删除：点击可删除该套餐
            -【套餐漏斗】页面
                -漏斗列表：漏斗名称、漏斗数量、创建时间、操作
                -操作：
                    -添加漏斗：侧边弹窗
                        -漏斗名称
                        -漏斗数量：单选（1、2、3、4）
                        -漏斗配置：根据选择的漏斗数量，确定需要配置的漏斗位数量
                            漏斗位#1：下拉选择（选择当前运营位已创建的套餐，支持模糊查询，需带出套餐ID）
                        -确定：关闭弹窗，列表成功新增一条目
                        -取消：关闭弹窗
                        -关闭：关闭弹窗
                    -编辑：侧边弹窗
                    -删除

        -新增tab【套餐运营策略】管理
            -套餐运营策略列表：策略名称、包含运营位（显示数量）、全局策略、操作
            -操作
                -添加策略：侧边弹窗
                    -策略名称
                    -运营位配置：可配置多个运营位
                        -选择运营位
                        -套餐配置：
                            -条件一：未购买云存储
                                -选择漏斗：下拉选择（漏斗中选择，单选）
                            -条件二：已购买云存储，3≤循环时长＜7
                                -选择漏斗：下拉选择（漏斗中选择，单选）
                            -条件三：已购买云存储，7≤循环时长＜15
                    -全局策略配置：
                        -每日弹窗上限（次/天）：录入文本，数值，非负数
                        -同类型弹窗上限（次/天）：录入文本，数值，非负数        
                    -确定：关闭弹窗，列表成功新增一条目
                    -取消：关闭弹窗
                    -关闭：关闭弹窗
                -编辑：侧边弹窗
                -删除

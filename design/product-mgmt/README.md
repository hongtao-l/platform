# 产品管理模块

> 管理物联网产品模型接入（以IPC为例）

---

## 功能概述

| 功能 | 说明 |
|------|------|
| 产品列表 | 产品增删改查 |
| 能力项配置 | 配置产品能力项（夜视模式、白光灯、红外灯等） |
| 参数设置 | 能力项参数配置 |

---

## 目录结构

```
design/product-mgmt/
├── README.md           # 模块说明
└── index.html          # 产品列表（主入口）
```

---

## 产品模型示例（IPC）

```
IPC产品
├── 基础信息
│   ├── 产品ID
│   ├── 产品名称
│   ├── 产品类型
│   └── 接入状态
│
└── 能力项
    ├── 夜视模式
    │   ├── 模式类型（自动/手动/关闭）
    │   └── 灵敏度参数
    ├── 白光灯
    │   ├── 亮度等级
    │   └── 自动开关
    ├── 红外灯
    │   ├── 照射距离
    │   └── 自动切换
    ├── 移动侦测
    │   ├── 侦测灵敏度
    │   └── 侦测区域
    └── 云存储
        └── 开关状态
```

## 业务背景

在不损害用户体验的前提下，寻找广告展示量的上限，并确保每一次展示都能产生最大的经济价值。如果没有这些控制，流量会因为"无效重复"而迅速贬值，导致广告主离场和用户流失。生态平台通过广告控制机制实现广告投放的精细化管理

---

## 名词解释

| 名词 | 解释 |
|---|---|
| 生态平台 | 面向运营人员的 Web 端后台管理系统，用于管理旗下 APP 的广告配置与用户策略 |
| APP | 接入生态平台的移动端应用 |
| 广告位 | APP 内预留的广告展示区域，分为开屏、插屏、Banner 等类型 |
| 广告源 | 广告位对应的具体广告投放来源，包含广告平台和广告源 ID |
| 广告策略 | 针对一组广告位的展示控制规则集合，包含频次、间隔、冷却等参数 |
| 用户分群 | 由数据仓库直接为用户打规则标签的用户集合，可绑定广告策略实现差异化投放 |
| 免广告用户 | 被授予免广告权益的特定用户，在有效期内不展示广告 |
| 广告ID | 广告源的唯一标识，生成规则：S + 5位随机数字，如 S47382 |
| 策略ID | 广告策略的唯一标识，生成规则：ad_日期(yyyyMMdd)_4位随机数字，如 ad_20260331_5821 |

## 业务流程图

### 广告配置流程

```
创建/选择 APP
    ↓
进入应用详情页
    ↓
【广告位管理 Tab】查看广告位列表
    ↓
选择广告位 → 进入广告源配置页
    ↓
添加/编辑广告源
    ├── 填写广告标题、选择广告平台、填写广告源ID
    └── 必填项校验，保存
    ↓
返回应用详情页
    ↓
【广告策略管理 Tab】管理广告策略
    ↓
添加/编辑广告策略
    ├── 填写策略名称
    ├── 配置广告位（选择广告位、广告源、展示状态、控制参数）
    ├── 配置全局策略（冷却间隔）
    └── 必填项校验，保存
```

### 用户分群与广告策略绑定流程

```
数据仓库为用户打规则标签
    ↓
分群数据定期同步至生态平台
    ↓
进入【广告用户分群】
    ↓
查看分群列表（分群名称、编码、用户数量由数仓同步）
    ↓
为分群配置广告策略
    ├── 点击【编辑】或【绑定策略】
    ├── 勾选需要配置的 APP
    ├── 选择对应的广告策略
    └── 保存配置
    ↓
分群绑定策略完成
```

### 免广告用户管理流程

```
进入【免广告用户】
    ↓
点击【添加用户】
    ↓
填写用户信息
    ├── 用户ID（必填）
    ├── 用户分群（必填）
    ├── 生效时长（可选）
    └── 备注（可选）
    ↓
确认添加，列表展示用户信息及有效期
```

## UI 交互规范

### 表单验证

**必填项标记：**
- 所有必填项在标签后显示红色星号 `*`
- 示例：`策略名称 *`

**错误提示：**
- 提交时校验所有必填项
- 未填写的必填项在字段下方显示红色错误提示："此项为必填项"
- 对应字段边框变红，提示用户
- 用户填写完整后，错误提示自动消失，字段边框恢复正常

**保存按钮状态：**
- 有必填项未填写时，保存按钮置灰不可点击
- 所有必填项填写完整后，保存按钮恢复可用

**实现细节：**
- 广告位/广告源/策略抽屉中，所有必填字段均需进行实时验证
- 用户在字段中输入内容时，如果之前有错误提示，输入完整后自动清除错误状态
- 广告策略配置中，若展示状态为"关闭"，则该广告位的控制参数（频次上限、展示间隔、刷新间隔）不需填写
- 若展示状态为"开启"，则这三个参数均为必填

### 删除操作

**删除限制提示：**
- 当数据被其他模块引用时，无法删除
- 点击删除按钮时，系统检查该数据是否被引用
- 若被引用，弹出"无法删除"提示弹窗，说明原因和解决方案
- 用户需先移除相关引用，才能删除该数据

**删除确认：**
- 可以删除的数据，点击删除按钮时显示确认对话框
- 用户确认后执行删除操作

**弹窗设计：**
- 标题："无法删除"
- 包含警告图标
- 显示具体的限制原因和解决方案
- 提供"我知道了"按钮关闭弹窗

### 删除操作

**删除限制提示：**
- 当数据被其他模块引用时，无法删除
- 点击删除按钮时，系统检查该数据是否被引用
- 若被引用，弹出"无法删除"提示弹窗，说明原因和解决方案
- 用户需先移除相关引用，才能删除该数据

**删除确认：**
- 可以删除的数据，点击删除按钮时显示确认对话框
- 用户确认后执行删除操作

### 侧边抽屉

**宽度规范：**
- 标准抽屉：480px（用于简单表单）
- 宽抽屉：600px（用于中等复杂度表单）
- 超宽抽屉：640px（用于复杂表单，如广告策略配置）

**交互规范：**
- 点击关闭按钮或背景遮罩关闭抽屉
- 抽屉打开时，背景显示半透明遮罩
- 抽屉有平滑的滑入/滑出动画

### 弹窗（Modal）

**用途：**
- 展示详情信息（如广告策略详情）
- 显示系统提示（如删除限制提示）

**交互规范：**
- 点击关闭按钮或背景遮罩关闭弹窗
- 弹窗有平滑的缩放和淡入/淡出动画
- 弹窗居中显示在屏幕中央

### 2. 应用管理 — 广告位管理 Tab

**列表字段：**

| 字段 | 说明 |
|---|---|
| 广告位名称 | 广告位的展示名称 |
| 广告位ID | 广告位唯一标识 |
| 广告类型 | 开屏 / 插屏 / Banner |
| 创建时间 | 广告位创建时间 |
| 操作 | 广告源配置、编辑、删除 |

**操作说明：**

- 广告源配置：点击后跳转至广告源配置三级页
- 编辑：点击【编辑】按钮，打开侧边抽屉编辑广告位信息
- 删除：点击【删除】按钮，检查是否被引用

**删除限制：**
- 广告位被广告策略引用时，无法删除
- 点击删除按钮时弹出提示："该广告位已被广告策略引用，无法删除。请先在广告策略中移除该广告位。"
- 用户需先编辑相关策略，移除对该广告位的引用，才能删除

**编辑广告位（侧边抽屉，宽度 480px）：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 广告位名称 | 文本输入 | 是 | 广告位的展示名称 |
| 广告位ID | 文本输入 | 是 | 广告位唯一标识，如 SLOT_001 |
| 广告类型 | 下拉选择 | 是 | 开屏 / 插屏 / Banner / 原生 / 激励视频 |

**校验规则：**
- 必填项校验：广告位名称、广告位ID、广告类型 均为必填
- 提交时校验：若有必填项未填写，在对应字段下方显示红色错误提示"此项为必填项"，保存按钮置灰不可点击
- 用户填写完整后，错误提示自动消失，保存按钮恢复可用

**删除限制：**

- 提交时校验：若有必填项未填写，在对应字段下方显示红色错误提示"此项为必填项"，保存按钮置灰不可点击

### 4. 应用管理 — 广告策略管理 Tab

**列表字段：**

| 字段 | 说明 |
|---|---|
| 策略名称 | 策略的展示名称 |
| 包含广告位 | 该策略配置的广告位数量 |
| 全局策略 | 展示开屏/插屏冷却间隔配置值 |
| 操作 | 编辑、删除 |

**删除限制：**
- 广告策略被用户分群引用时，无法删除，点击删除按钮时弹出提示："该广告策略已被用户分群引用，无法删除。请先在用户分群中移除该策略的绑定。"

**添加/编辑广告策略（侧边抽屉，宽度 640px）：**

**基础信息：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 策略名称 | 文本输入 | 是 | 策略展示名称 |

**配置广告位（可动态添加/删除，不可重复选择同一广告位）：**

每个广告位配置块包含：

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 选择广告位 | 下拉选择 | 是 | 从当前 APP 广告位中选择，不可重复 |
| 广告源 | 下拉选择 | 是 | 选择该广告位对应的广告源 |
| 展示状态 | 开关（Toggle） | 是 | 开启后需配置以下控制参数 |
| 频次上限（次/小时） | 数字输入 | 条件必填 | 展示状态开启时必填 |
| 展示间隔（s） | 数字输入 | 条件必填 | 展示状态开启时必填 |
| 刷新间隔（s） | 数字输入 | 条件必填 | 展示状态开启时必填 |

重复校验规则：同一策略内，同一广告位只能配置一次；选择重复时，下拉框标红并显示错误提示"该广告位已被添加，请勿重复配置"；点击保存时再次校验，有重复则阻止提交。

**配置全局策略：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 开屏/插屏冷却间隔（s） | 数字输入 | 是 | 开屏广告与插屏广告之间的最小间隔时间 |

**校验规则：**
- 必填项校验：策略名称、至少一个广告位配置、每个广告位的广告源、展示状态开启时的控制参数（频次上限、展示间隔、刷新间隔）、开屏/插屏冷却间隔均为必填
- 提交时校验：若有必填项未填写，在对应字段下方显示红色错误提示"此项为必填项"，保存按钮置灰不可点击
- 用户填写完整后，错误提示自动消失，保存按钮恢复可用
- 广告位配置块校验：至少需要配置一个广告位，否则显示提示"请至少配置一个广告位"

### 删除限制规则（全局）

为了保证数据的完整性和一致性，系统对广告位、广告源、广告策略的删除进行了限制：

**广告位删除限制：**
- 当广告位被任何广告策略引用时，无法删除
- 点击删除按钮时，系统检查该广告位是否被引用
- 若被引用，弹出提示："该广告位已被广告策略引用，无法删除。请先在广告策略中移除该广告位。"
- 用户需先编辑相关策略，移除对该广告位的引用，才能删除

**广告源删除限制：**
- 当广告源被任何广告策略引用时，无法删除
- 点击删除按钮时，系统检查该广告源是否被引用
- 若被引用，弹出提示："该广告源已被广告策略引用，无法删除。请先在广告策略中移除该广告源。"
- 用户需先编辑相关策略，移除对该广告源的引用，才能删除

**广告策略删除限制：**
- 当广告策略被任何用户分群绑定时，无法删除
- 点击删除按钮时，系统检查该策略是否被分群引用
- 若被引用，弹出提示："该广告策略已被用户分群引用，无法删除。请先在用户分群中移除该策略的绑定。"
- 用户需先编辑相关分群，移除对该策略的绑定，才能删除

### 5. 用户管理 — 广告用户分群

**页面路径：** 用户管理 / 广告用户分群

**设计说明：**
用户分群由数据仓库直接为用户打规则标签，分群规则由开发写死配置。生态平台作为数据展示和策略配置的入口，运营人员可在此页面查看分群信息并为各分群配置广告策略。

**列表字段：**

| 字段 | 说明 |
|---|---|
| 序号 | 列表序号 |
| 分群名称 | 分群展示名称（由数据仓库同步） |
| 分群编码 | 分群唯一标识（由数据仓库同步） |
| 用户数量 | 当前分群内的用户数（由数据仓库同步） |
| 广告策略 | 展示当前绑定的策略状态；已绑定时显示【查看】和【重新绑定】按钮；未绑定时显示"未绑定"及【绑定策略】按钮 |

**查看分群绑定策略（侧边抽屉，宽度 600px）：**

点击【查看】按钮弹出，展示：
- 当前分群绑定的广告策略配置（仅查看）
- 对于已绑定策略的 APP，在其下方展示该策略的详细信息：
  - 全局策略：开屏/插屏冷却间隔
  - 广告位配置明细表（广告位名称、广告源名称、展示状态、频次上限、展示间隔、刷新间隔）
- 对于未绑定的 APP，仅显示 APP 信息和"未绑定"状态
- 仅支持查看，不支持编辑
- 底部仅有【关闭】按钮

**编辑分群（侧边抽屉，宽度 680px）：**

由于分群规则由开发写死，编辑功能仅支持配置广告策略绑定，不支持修改分群规则。

**广告策略绑定：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| APP 勾选 | 多选复选框 | 否 | 勾选需要配置广告策略的 APP，支持全选 |
| 广告策略 | 下拉选择（单选） | 条件必填 | 勾选 APP 后必须选择对应策略；选择后展示策略内容预览（策略名称、ID、冷却间隔、广告位配置明细，包含广告源信息） |

**删除限制：**
- 分群数据由数据仓库同步，删除操作需谨慎
- 分群被广告策略绑定时，无法删除，点击删除按钮时弹出提示："该分群已被广告策略绑定，无法删除。请先移除该分群的策略绑定。"

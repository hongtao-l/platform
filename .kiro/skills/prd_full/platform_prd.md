## 业务背景

生态平台 Web 管理系统旨在为运营人员提供一站式的套餐运营管理能力。系统涵盖套餐管理、商城位运营、推荐位运营三大核心模块，通过配置套餐池、运营策略和用户分群，实现对不同用户群体的差异化套餐推送，提升套餐转化率和用户付费意愿。

系统支持多应用管理，运营人员可针对不同应用（如牵心PRO、鹤梦之家）配置独立的套餐运营策略，实现精细化的运营管理。

## 名词解释

| 名词 | 解释 |
|---|---|
| 套餐 | 向用户提供的付费服务组合，包含云存储、AI服务等类型 |
| 套餐池 | 与运营位绑定的套餐集合，支持分类管理 |
| 商城位 | APP 内商城页面的套餐展示区域 |
| 推荐位 | APP 内推荐场景的套餐展示区域 |
| 运营位编码 | 运营位的唯一标识，不可重复 |
| 漏斗 | 套餐展示的排序规则，包含价格漏斗和行为漏斗两种类型 |
| 漏斗位 | 漏斗中的具体套餐展示位置，固定4个位置 |
| 商城位策略 | 针对商城位的套餐推送规则集合 |
| 推荐位策略 | 针对推荐位的套餐推送规则集合，包含条件漏斗配置 |
| 默认策略 | 未命中用户分群时展示的策略，投放区域默认为国内+海外，投放用户分群为空 |
| 投放区域 | 套餐推送的目标区域，支持国内、海外多选 |
| 投放用户分群 | 套餐推送的目标用户群体 |
| 弹窗间隔 | 两次弹窗之间的最小间隔时间（小时） |
| 订阅制 | 用户按周期自动续费的套餐类型 |
| 非订阅 | 用户一次性购买的套餐类型 |
| 免费试用 | 订阅制套餐的试用功能，支持配置试用周期和过滤条件 |

## 业务流程图

### 套餐管理流程

```
进入套餐管理页面
    ↓
查看套餐池列表（按分类展示）
    ↓
创建套餐
    ├── 填写基础信息（名称、推荐标题、推荐文案）
    ├── 选择套餐服务（应用方向、生效时长、套餐服务）
    ├── 配置内购Key（Apple Key、Google Key）
    ├── 选择订阅类型
    │   ├── 非订阅：配置划线价、售价
    │   └── 订阅制：配置订阅文案、划线价、免费试用、订阅售价
    ├── 选择套餐标签
    └── 选择套餐描述标签
    ↓
套餐创建成功，加入套餐池
    ↓
支持编辑、删除、上下架操作
```

### 商城位运营流程

```
选择应用（牵心PRO / 鹤梦之家）
    ↓
查看商城位策略列表
    ↓
添加策略
    ├── Step 1: 基础信息（策略名称、备注）
    └── Step 2: 投放配置（投放区域、投放用户分群）
    ↓
策略创建成功
    ↓
进入套餐配置页面
    ├── 添加套餐（从套餐池选择）
    ├── 调整排序（拖拽）
    └── 设置上架状态（开关）
```

### 推荐位运营流程

```
选择应用及推荐位
    ↓
【推荐位套餐漏斗】管理漏斗
    ├── 添加漏斗（名称、类型、4个漏斗位套餐）
    ├── 编辑漏斗
    └── 删除漏斗
    ↓
【推荐位策略】管理策略
    ├── Step 1: 基础信息（策略名称、落地页URL、备注）
    ├── Step 2: 漏斗配置（条件一/二/三选择漏斗、弹窗间隔）
    └── Step 3: 投放配置（投放区域、投放用户分群）
    ↓
策略创建成功
```

### 用户分群绑定策略流程

```
进入用户分群页面
    ↓
选择分群类型（商城用户分群 / 推荐用户分群）
    ↓
查看分群列表
    ↓
点击【绑定策略】
    ↓
选择APP（多选）
    ↓
选择对应APP下的运营策略（单选）
    ↓
查看策略详细内容确认
    ↓
保存绑定
```

## UI 交互规范

### 表单验证

**必填项标记：**
- 所有必填项在标签后显示红色星号 `*`

**错误提示：**
- 提交时校验所有必填项
- 未填写的必填项在字段下方显示红色错误提示："此项为必填项"
- 对应字段边框变红
- 用户填写完整后，错误提示自动消失，字段边框恢复正常

**保存按钮状态：**
- 有必填项未填写时，保存按钮置灰不可点击
- 所有必填项填写完整后，保存按钮恢复可用

### 删除操作

**删除限制提示：**
- 当数据被其他模块引用时，无法删除
- 若被引用，弹出"无法删除"提示弹窗，说明原因和解决方案

**删除确认：**
- 可以删除的数据，点击删除按钮时显示确认对话框，用户确认后执行删除

### 侧边抽屉

**宽度规范：**
- 标准抽屉：480px（简单表单）
- 宽抽屉：560px（中等复杂度表单）
- 超宽抽屉：640px（复杂表单）
- 步骤式抽屉：720px（多步骤表单）

### 列表添加/编辑弹窗规则

所有列表的添加弹窗和编辑弹窗内容完全一致，唯一区别是弹窗标题文案：
- 添加：**添加 XX**
- 编辑：**编辑 XX**

### 步骤式表单规范

**步骤条样式：**
- 当前步骤：蓝色圆形背景 + 白色数字 + 蓝色标签
- 未完成步骤：灰色圆形背景 + 灰色数字 + 灰色标签
- 已完成步骤：蓝色圆形背景 + 白色数字 + 深色标签
- 步骤间连线：未完成时灰色，完成时蓝色

**按钮规则：**
- 第一步：显示【取消】【下一步】
- 中间步骤：显示【上一步】【取消】【下一步】
- 最后一步：显示【上一步】【取消】【提交】

## 详细功能说明

### 1. 套餐管理

**页面路径：** 应用管理 / {APP名称} / {运营位名称} · 套餐管理

**页面结构：**

顶部展示面包屑导航、返回按钮及运营位信息，下方为套餐池列表，按分类（云存储套餐、AI智能服务、综合套餐）展示。

**列表字段：**

| 字段 | 说明 |
|---|---|
| 拖拽手柄 | 用于拖拽调整分类内排序 |
| 套餐信息 | 套餐名称 + 套餐ID |
| 应用方向 | 设备 / 用户 |
| 订阅类型 | 订阅制 / 非订阅 |
| 售价 | 人民币 / 美元格式，如 ¥6.90 / $0.95 |
| 上架状态 | 开关控件，支持快速切换 |
| 操作 | 移除 |

**操作说明：**

- 创建套餐：点击【创建套餐】按钮，弹出侧边抽屉（宽度 640px）
- 移除：点击可将该套餐从套餐池中移除，移除前显示确认对话框
- 上架状态：通过开关控件快速切换上架/下架状态
- 拖拽排序：通过拖拽手柄调整分类内套餐排序

**创建/编辑套餐（侧边抽屉，宽度 640px）：**

**基础信息：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 套餐名称 | 文本输入 | 是 | 套餐展示名称 |
| 推荐标题 | 文本输入 | 是 | 套餐推荐标题 |
| 推荐文案 | 文本输入 | 是 | 套餐推荐文案 |

**选择套餐服务：**

| 字段 | 类型 | 必填 | 说明 |

| 应用方向 | 单选 | 是 | 设备套餐 / 用户套餐 |

**设备套餐：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 生效时长 | 下拉选择 | 是 | 7天、15天、30天、1年 |
| 套餐服务 | 下拉选择 | 是 | 7天事件云存、15天事件云存等 |
| 套餐服务详情 | 只读展示 | — | 套餐服务名称、服务类型、可用设备上限、平台成本 |

**用户套餐：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 生效时长 | 下拉选择 | 是 | 3个月、6个月 |
| 套餐服务 | 下拉选择 | 是 | 短信套餐、免广告-年等 |
| 套餐服务详情 | 只读展示 | — | 套餐服务名称、服务类型、平台成本 |

**内购Key配置（选填）：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| Apple Key | 文本输入 | 否 | Apple 内购 Key |
| Google Key | 文本输入 | 否 | Google 内购 Key |

**订阅类型：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 订阅类型 | 单选 | 是 | 非订阅 / 订阅制 |

**非订阅：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 划线价（¥） | 数字输入 | 是 | 原价，人民币 |
| 划线价（$） | 数字输入 | 是 | 原价，美元 |
| 售价（¥） | 数字输入 | 是 | 实际售价，人民币 |
| 售价（$） | 数字输入 | 是 | 实际售价，美元 |

**订阅制：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 订阅文案 | 文本输入 | 是 | 如：每月自动续费，可随时取消 |
| 划线价（¥） | 数字输入 | 否 | 原价，人民币 |
| 划线价（$） | 数字输入 | 否 | 原价，美元 |

| 试用周期（天） | 数字输入 | 条件必填 | 免费试用开启时必填 |
| 过滤条件 | 下拉选择 | 条件必填 | 从未试用过此套餐 / 从未试用过该服务类型 / 不限制 |

| 订阅售价（$） | 数字输入 | 是 | 订阅价格，美元 |

**套餐标签：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 套餐标签 | 多选 | 是 | 免费试用、限时优惠、新用户特惠 |

**套餐描述标签：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 套餐描述标签 | 多选 | 是 | 7天循环事件录制、7天消息回溯等 |

**校验规则：**
- 所有必填项未填写时，显示错误提示

- 套餐编码不可与已有编码重复

### 2. 商城位运营

**页面路径：** 运营管理 / 商城位运营

**页面结构：**

左侧为应用列表（牵心PRO、鹤梦之家），右侧为当前应用的商城位策略列表。

**应用列表：**

| 字段 | 说明 |
|---|---|
| 应用图标 | 应用图标 |
| 应用名称 | 牵心PRO / 鹤梦之家 |
| 应用ID | QX_PRO_001 / HM_HOME_001 |

**商城位策略列表：**

| 字段 | 说明 |
|---|---|
| 策略名称 | 策略展示名称 |
| 投放区域 | 国内 / 海外，支持点击直接变更 |
| 投放用户分群 | 已绑定的用户分群，点击可编辑 |
| 套餐配置 | 显示套餐数量，点击进入套餐配置页面 |
| 备注 | 策略备注信息 |
| 操作 | 编辑、删除 |

**默认策略：**
- 每个应用存在一个默认策略，背景色为浅黄色
- 默认策略投放区域默认为国内+海外，投放用户分群为空
- 默认策略不支持删除，仅支持编辑
- 编辑默认策略时，投放区域和投放用户分群字段不可编辑

**添加策略（步骤式侧边抽屉，宽度 720px）：**

**Step 1: 基础信息**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 策略名称 | 文本输入 | 是 | 策略展示名称 |
| 备注 | 文本输入 | 否 | 策略备注信息 |

**Step 2: 投放配置**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 投放区域 | 多选 | 是 | 国内、海外 |
| 投放用户分群 | 多选 | 是 | 高价值用户、新注册用户、活跃用户、沉默用户、付费用户等 |

**编辑策略：**
- 编辑弹窗内容同添加一致
- 默认策略编辑时，投放区域和投放用户分群字段置灰不可编辑

**删除限制：**
- 策略被用户分群绑定时，无法删除
- 弹出提示："该策略已被用户分群引用，无法删除。请先在用户分群中移除该策略的绑定。"

## 删除限制规则（全局）

**套餐删除限制：**
- 套餐被策略引用时，无法删除
- 弹出提示："该套餐已被策略引用，无法删除。请先在策略中移除该套餐。"

**漏斗删除限制：**
- 漏斗被策略引用时，无法删除
- 弹出提示："该漏斗已被策略引用，无法删除。请先在策略中移除该漏斗。"

**策略删除限制：**
- 策略被用户分群绑定时，无法删除
- 弹出提示："该策略已被用户分群引用，无法删除。请先在用户分群中移除该策略的绑定。"

**默认策略删除限制：**
- 默认策略不支持删除操作

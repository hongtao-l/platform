# CareCam Pro APP 设计文档说明

## 一、项目概述

**项目名称**: CareCam Pro - 智能摄像头管理APP

**项目目标**: 打造一款专业的IPC（监控）设备管理应用，支持设备管理、实时预览、消息通知、云存储服务、AI智能服务等功能。

**技术栈**: Vue3 + Vite + JavaScript + Vant 4

**设计尺寸**: 375 x 812 px (iPhone X标准尺寸)

---

## 二、页面结构

### 2.1 主要页面

| 页面 | 路由 | 说明 |
|------|------|------|
| 首页 | /home | 设备列表、家庭分组、快捷操作 |
| 消息 | /messages | AI搜索、消息列表、筛选过滤 |
| 我的 | /profile | 个人中心、服务商城入口 |
| 服务商城 | /store | 云存储、AI服务、流量充值等 |
| 实时预览 | /liveview | 视频播放、控制面板 |
| 回放 | /playback | 云回放/SD卡回放切换 |
| 设备设置 | /settings | 设备参数配置 |
| 电池管理 | /battery | 工作模式、电量统计 |
| 智能模式 | /smart-schedule | 时间计划配置 |

### 2.2 底部导航

首页、消息、我的 三个Tab页面显示底部导航栏，其他页面隐藏。

## 三、功能模块详细设计

### 3.1 首页 (Home)

#### 3.1.1 顶部栏
- **左侧**: 家庭名称 + 设备状态统计（在线/离线/休眠数量）
- **右侧**: 
  - 消息按钮（带红点提示）
  - 添加设备按钮
  - 用户头像（点击进入个人中心）

#### 3.1.2 家庭分组标签
- 横向滚动标签栏
- 默认分组: 全部家庭、客厅、入口、后院
- 支持添加新分组

#### 3.1.3 设备卡片
每个设备卡片包含:
- **缩略图区域**: 16:9比例，显示设备画面或占位图
- **状态标签**: 在线(绿色)、离线(灰色)、休眠(黄色)
- **分享标签**: 已分享设备显示蓝色标签
- **设备名称**: 显示在缩略图底部
- **操作按钮**: 消息、回放、云存储状态、设置

#### 3.1.4 添加设备
- 虚线边框卡片
- 点击弹出添加设备引导弹窗

### 3.2 消息页 (Messages)

#### 3.2.1 AI搜索栏

- **设备筛选**: 全部设备 / 指定设备
- **事件类型**: 全部事件、人形、移动、车辆、声音、包裹、动物

#### 3.2.3 消息列表
- 消息卡片: 缩略图 + 事件类型 + 设备名称 + 时间 + 标签
- 点击进入消息详情页

#### 3.2.4 底部营销提示
- 服务推荐入口

- 未激活服务提示

### 3.8 电池管理 (Battery Management)

#### 3.8.1 电池状态
- 电量百分比圆环
- 电池状态（良好/一般/较差）
- 预计续航时间
- 电量进度条

#### 3.8.2 工作模式

| 模式 | 说明 | 配置项 |
|------|------|--------|
| 省电模式 | 默认休眠，事件唤醒录像 | 无 |
| AOV模式 | 持续运行，定时抓拍 | 拍照帧率、省电阈值 |
| 常电模式 | 连续录像 | 省电阈值 |
| 智能模式 | 按时间计划切换模式 | 时间计划配置 |

#### 3.8.3 AOV模式配置
- **拍照帧率**: 1-10秒/帧（数值越小功耗越高）
- **省电模式电量阈值**: 10%-50%，低于阈值自动切换省电模式

#### 3.8.4 事件录像设置
- **录像时长**: 5-30秒滑块
- **运动停止则停止录像**: 开关

#### 3.8.5 运行统计
- 近24小时电量趋势图
- 过去10天电量消耗图

## 四、营销弹窗场景

### 4.1 场景列表

| 场景 | 触发条件 | 推荐套餐 |
|------|----------|----------|
| sd_full | SD卡存储将满(92%) | 30天云存储 |
| cloud_expire | 云存即将届满 | 30天云存储 |
| cloud_no_service | 无云服务 | 7天云存储 |
| no_sd | 未检测到SD卡 | 7天云存储 |
| msg_no_service | 消息页无云服务 | 7天云存储 |
| msg_expire | 消息记录即将届满 | 30天云存储 |

### 4.2 弹窗内容
- 图标 + 标题 + 描述
- 推荐套餐卡片（折扣标签、套餐名、价格）
- 支付方式选择（支付宝/微信）
- 操作按钮（主按钮 + 次按钮）
